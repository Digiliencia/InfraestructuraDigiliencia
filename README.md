# Proyecto DIGILIENCIA

## Variables de entorno
Las variables de entorno están definidas en el fichero `.env` en el directorio raíz del proyecto y son las siguientes:

```bash
DDBB_USERNAME   # Usuario de la base de datos
DDBB_PASSWORD   # Contraseña de la base de datos
DDBB_URI        # URI de la base de datos
```
