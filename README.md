# Proyecto DIGILIENCIA

## Variables de entorno

Las variables de entorno están definidas en el fichero `.env` en el directorio raíz del proyecto y son las siguientes:

```bash
DDBB_URI                # URI de la base de datos Neo4j

WEFORUM_EMAIL           # Correo electrónico utilizado para la autenticación en WeForum
WEFORUM_PASSWD          # Contraseña utilizada para la autenticación en WeForum
WEBDRIVERWAIT_TIMEOUT   # Tiempo de espera para el WebDriver en segundos

LLM_URL                 # URL de la API del LLM (preparado para Ollama)
CLASSIFICATION_MODEL    # LLM usado para la clasificación de topics
```
