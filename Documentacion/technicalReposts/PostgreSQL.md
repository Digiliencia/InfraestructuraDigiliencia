# PostgreSQL

La base de datos SQL se levantará sobre PostgreSQL. La elección del gestor de bases de datos obedece a las razones de escalabilidad, cumplimiento de las características ACID, facilidad de uso, empleo de SQL estándar, amplia comunidad y *open source*.

Otras soluciones requieren licencias comerciales o su escalabilidad o comunidad son inferiores a las del gestor elegido.
