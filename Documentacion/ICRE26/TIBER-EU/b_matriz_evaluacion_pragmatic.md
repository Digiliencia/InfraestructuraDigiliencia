# Matriz de Evaluación PRAGMATIC Completa  
## Para indicadores TIBER‑EU / TIBER‑ES

---

## 1. Escala de puntuación

Cada criterio PRAGMATIC se puntúa de 0 a 5:

- 0–1: Muy pobre.  
- 2–3: Aceptable / moderado.  
- 4–5: Alto / excelente.

## 3. Uso práctico de la matriz

- Calcular para cada métrica una **puntuación global PRAGMATIC**, por ejemplo la media de los nueve criterios.  
- Definir umbrales para clasificación:  
  - ≥ 4: métrica “estrella”.  
  - 3–4: métrica “útil”.  
  - < 3: métrica “opcional” o candidata a revisión.

- Visualizar el resultado en gráficos radar por métrica o por dimensión para priorizar.
