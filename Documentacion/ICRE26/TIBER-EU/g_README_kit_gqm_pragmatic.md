# README del Kit GQM + PRAGMATIC para Indicadores TIBER

---

## 1. Qué es este kit

Conjunto de documentos para:

- Diseñar métricas TIBER‑EU / TIBER‑ES trazables a objetivos nacionales.  
- Evaluarlas con el marco PRAGMATIC.  
- Integrarlas en encuestas, hojas Excel y reportes ejecutivos.

---

## 2. Contenido

- `a_marco_integrativo_gqm_pragmatic.md` – Definición del marco y de las metas/métricas.  
- `b_matriz_evaluacion_pragmatic.md` – Puntuación de cada métrica según PRAGMATIC.  
- `c_narrativa_gqm_pragmatic.md` – Texto para explicar el enfoque a públicos no técnicos.  
- `d_mapeo_preguntas_normativa_gqm.md` – Tabla que enlaza preguntas, metas, métricas y normas.  
- `e_plantilla_excel_igm_roi_gqm.md` – Guía para implementar cálculos en Excel.  
- `f_plantilla_reporte_ejecutivo_gqm.md` – Guion de presentación para dirección/reguladores.

---

## 3. Cómo usarlo

1. **Leer el marco (a)** para entender las dimensiones y metas.  
2. **Seleccionar métricas** con ayuda de la matriz PRAGMATIC (b).  
3. **Alinear la encuesta** actualizando el mapeo normativo (d).  
4. **Construir o adaptar la hoja Excel** usando (e).  
5. **Comunicar resultados** con la plantilla de presentación (f) y la narrativa (c).

---

## 4. Actualización y mantenimiento

- Revisar al menos cada dos años o tras cambios relevantes en TIBER‑EU/TIBER‑ES/DORA.  
- Documentar cualquier cambio de ponderaciones PRAGMATIC o inclusión/exclusión de métricas.
