# README del Kit GQM + PRAGMATIC para indicadores SQUARE  
## Versión 1.0

---

## 1. ¿Qué es este kit?

Este kit amplía el kit de encuesta SQUARE para requisitos de seguridad incorporando:

- La metodología **GQM (Goal–Question–Metric)**.  
- La evaluación **PRAGMATIC** de calidad de métricas.

El objetivo es asegurar que los indicadores usados a nivel organizativo y nacional:

- Nacen de objetivos claros.  
- Responden preguntas relevantes.  
- Superan un mínimo de higiene métrica.

---

## 2. Contenidos

1. `marco_gqm_pragmatic_square.md`  
   Define el marco integrativo GQM + PRAGMATIC aplicado a indicadores SQUARE.

2. `matriz_pragmatic_square.md`  
   Contiene la matriz PRAGMATIC completa para las principales métricas.

3. `narrativa_gqm_pragmatic_square.md`  
   Explica el porqué del enfoque y cómo usarlo en la práctica.

4. `mapeo_indicadores_normativa_square.md`  
   Relaciona cada métrica con objetivos GQM y marcos normativos relevantes.

5. `plantilla_excel_gqm_pragmatic_square.md`  
   Especifica la estructura de un Excel para calcular IGM, métricas y ROI.

6. `plantilla_reporte_gqm_pragmatic_square.md`  
   Ofrece una estructura de presentación ejecutiva.

---

## 3. Cómo usarlo

1. **Lea el marco integrativo** para entender la lógica GQM + PRAGMATIC.  
2. **Revise la matriz PRAGMATIC** y ajuste, si es preciso, las valoraciones para su sector.  
3. **Implemente la plantilla Excel** o un modelo equivalente en su herramienta preferida.  
4. **Integre las métricas** con la encuesta SQUARE ya aplicada o en curso.  
5. **Use la plantilla de reporte** para comunicar resultados.

---

## 4. Escenarios de aplicación

- **Dentro de una organización**: para elegir un conjunto razonable de métricas de requisitos de seguridad y justificarlo.  
- **En un programa sectorial**: para garantizar comparabilidad entre múltiples entidades.  
- **A nivel nacional**: para apoyar la evaluación de la Estrategia Nacional de Ciberseguridad desde la perspectiva de diseño seguro.

---

## 5. Principios de uso responsable

- No utilice las métricas como arma arrojadiza, sino como instrumento de mejora.  
- Sea transparente sobre las limitaciones de las métricas, especialmente en el ámbito económico.  
- Revise periódicamente las valoraciones PRAGMATIC a la luz de la experiencia y nuevos datos.  
- Evite inflar cuadros de mando; mejor pocas métricas buenas que muchos números decorativos.
