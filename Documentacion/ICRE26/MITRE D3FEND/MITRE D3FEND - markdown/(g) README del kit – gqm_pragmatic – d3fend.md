# (g) README del Kit GQM + PRAGMATIC para indicadores MITRE D3FEND

## 1. Contenido

Este kit complementa la encuesta MITRE D3FEND con un marco de indicadores basado en GQM y evaluado con los criterios PRAGMATIC:

- (a) Marco integrativo GQM + PRAGMATIC.
- (b) Matriz de Evaluación PRAGMATIC Completa.
- (c) Narrativa explicativa.
- (d) Mapeo métricas–normativa.
- (e) Plantilla de Excel para IGM y ROI.
- (f) Plantilla de reporte ejecutivo.

## 2. Uso recomendado

1. Seleccionar las métricas M1.x–M5.x más adecuadas para el contexto nacional o sectorial.[web:5][web:18]
2. Ajustar los pesos del IGM y las puntuaciones PRAGMATIC según prioridades locales.[web:18][web:21]
3. Integrar estas métricas en la encuesta y en los sistemas de recogida de datos.
4. Utilizar la plantilla de Excel para calcular IGM y ROI de iniciativas ligadas a D3FEND.[web:5][web:18][web:21]
5. Comunicar resultados mediante el reporte ejecutivo a alta dirección y reguladores.[web:25][web:26][web:27]

## 3. Adaptación y extensión

El marco está diseñado para evolucionar: se pueden añadir nuevas métricas (por ejemplo, relacionadas con desinformación, identidad digital o IA) siguiendo el mismo esquema GQM + PRAGMATIC.

## 4. Espíritu del kit

El objetivo no es producir más dashboards, sino mejores conversaciones: conversaciones donde “cobertura”, “latencia” o “resiliencia” se expresan en números claros, rastreables hasta amenazas concretas y justificables ante quienes deciden dónde se invierte el próximo euro en seguridad.
