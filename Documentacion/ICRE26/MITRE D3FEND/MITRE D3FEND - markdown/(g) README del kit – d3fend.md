# (g) README del Kit de Encuesta MITRE D3FEND

## 1. Contenido del kit

Este kit contiene los siguientes componentes, todos en formato Markdown para facilitar su adaptación:

- (a) Modelo de encuesta integral.
- (b) Guía metodológica detallada.
- (c) Narrativa explicativa.
- (d) Mapeo de preguntas a requisitos normativos.
- (e) Plantilla de Excel para cálculo de IGM y ROI (descripción estructurada).
- (f) Plantilla de reporte ejecutivo en PowerPoint.

## 2. Público objetivo

- Responsables de ciberseguridad (CISOs, responsables de SOC/CSIRT).
- Responsables de continuidad, resiliencia y riesgo.
- Organismos reguladores y supervisores sectoriales.
- Equipos de investigación y think tanks especializados en ciberseguridad.

## 3. Cómo utilizar el kit

1. Adaptar el modelo de encuesta a la realidad sectorial o nacional (terminología, referencias regulatorias).[web:25][web:26][web:27]
2. Implementar la encuesta en la herramienta elegida (plataforma online, formulario interno, etc.).
3. Utilizar la guía metodológica para definir el plan de análisis y segmentación.
4. Configurar el libro de Excel siguiendo la plantilla descrita, para calcular el IGM y estimar el ROI.[web:18][web:21]
5. Emplear la plantilla de reporte ejecutivo para comunicar los resultados a alta dirección, reguladores o foros sectoriales.[web:5]

## 4. Recomendaciones de implementación

- Asegurar la confidencialidad y anonimato de las respuestas.
- Comunicar con transparencia el propósito del ejercicio y el uso previsto de los resultados.
- Programar la encuesta con una periodicidad razonable (por ejemplo, anual) para seguir la evolución.
- Complementar la encuesta con ejemplos prácticos de modelado D3FEND CAD en talleres o ejercicios.[web:19][web:29]

## 5. Licencia y adaptación

El kit está diseñado para ser adaptado, extendido o recortado sin remordimientos, siempre que se mantenga el espíritu original: utilizar D3FEND y ATT&CK como lenguajes comunes para medir, con cierta precisión, qué sabemos defender y qué seguimos dejando al azar.[web:1][web:3][web:5]
