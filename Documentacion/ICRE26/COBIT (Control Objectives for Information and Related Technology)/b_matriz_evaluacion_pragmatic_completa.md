# Matriz de Evaluación PRAGMATIC Completa para indicadores COBIT

> Matriz base para evaluar y ajustar la calidad de las métricas propuestas.[web:61][web:64]

---

## 1. Instrucciones de uso

1. Cada fila corresponde a una **métrica** (M1.1, M1.2, etc.).  
2. Cada columna de criterio PRAGMATIC se puntúa de 0 a 10.  
3. La columna `Score_Total` puede ser la suma o la media de los nueve criterios.  
4. Se recomienda fijar un umbral mínimo (p.ej. 60/90) para considerar una métrica “apta para producción”.

## 3. Personalización

- Ajustar los scores a la realidad y prioridades del país o sector.  
- Añadir métricas nuevas o variantes sectoriales (p.ej. métricas específicas para sanidad, energía, financiero).  
- Usar el `Score_Total` para priorizar qué métricas entrarían en paneles nacionales o sectoriales.
