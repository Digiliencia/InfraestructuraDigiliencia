# README – Kit GQM + PRAGMATIC para indicadores FAIR nacionales

## 1. Contenido del kit

Este kit incluye varios componentes en formato Markdown diseñados para construir y gobernar un sistema de indicadores FAIR a nivel nacional o sectorial:

- Marco integrativo GQM + PRAGMATIC.
- Matriz de Evaluación PRAGMATIC de los indicadores.
- Narrativa explicativa para decisores.
- Mapeo de indicadores/preguntas a requisitos normativos.
- Plantilla de Excel para cálculo de IGM/ROI alineada con GQM.
- Plantilla de reporte ejecutivo para foros de alto nivel.

## 2. Público objetivo

- Responsables de estrategias nacionales de ciberseguridad.
- Reguladores y supervisores sectoriales.
- Equipos de inteligencia de ciber‑riesgo y analistas macroprudenciales.
- Consorcios sectoriales y asociaciones empresariales.

## 3. Uso recomendado

1. Ajustar el marco GQM a los objetivos concretos del país o sector.
2. Seleccionar el subconjunto de indicadores FAIR que se considere viable construir en una primera fase.
3. Completar la matriz PRAGMATIC con puntuaciones acordadas por un grupo multidisciplinar.
4. Diseñar la recogida de datos (encuestas, CSIRTs, aseguradoras, etc.) alineada con las definiciones de métricas.
5. Implementar la plantilla de Excel, adaptada a los datos disponibles.
6. Usar la plantilla de reporte ejecutivo para presentar resultados y decisiones.

## 4. Extensiones posibles

- Incorporar métricas adicionales específicas de OT, IA o desinformación.
- Ampliar el mapeo normativo con referencias legales nacionales detalladas.
- Integrar el kit en una plataforma de datos oficial o en un observatorio nacional.

## 5. Advertencia conceptual

Este kit no pretende resolver todos los debates metodológicos sobre cuantificación de ciber‑riesgo, pero sí ofrece una forma estructurada de evitar dos extremos igualmente peligrosos: la parálisis por análisis y el entusiasmo por métricas poco útiles. El uso disciplinado de GQM y PRAGMATIC es la mejor defensa contra ambos.
