# README – Kit de Encuesta FAIR de Ciber‑Riesgo

## 1. Contenido del kit

Este kit incluye los siguientes componentes en formato Markdown:

- Modelo de encuesta integral.
- Guía metodológica detallada.
- Narrativa explicativa para acompañar resultados.
- Mapeo básico de preguntas a referencias normativas.
- Plantilla conceptual para cálculo de IGM y ROI en Excel.
- Plantilla de reporte ejecutivo tipo presentación.

## 2. Uso recomendado

1. Adaptar el modelo de encuesta a las particularidades del sector o país (sin traicionar el espíritu de FAIR).
2. Implementar la encuesta en una herramienta online o integrarla en el sistema de gestión de riesgos existente.
3. Utilizar la guía metodológica para formar a los responsables que responderán la encuesta.
4. Recopilar y analizar resultados, construyendo indicadores de madurez y exposición.
5. Emplear la plantilla de Excel para traducir algunos resultados en cálculos de IGM y ROI de controles.
6. Presentar hallazgos a la alta dirección con la plantilla de reporte ejecutivo.

## 3. Personalización y ampliación

El kit está pensado como base común. Es esperable y saludable que cada organización o consorcio:

- Añada preguntas específicas de su sector (p. ej., energía, salud, finanzas).
- Profundice en el mapeo normativo para incluir referencias locales.
- Enriquezca la plantilla de Excel con datos internos y modelos propios.
