# (g) README del Kit GQM + PRAGMATIC para indicadores ISO/IEC 42001 – Ciberseguridad de IA

## 1. Contenido del kit

Este kit complementa el modelo de encuesta ISO/IEC 42001 incorporando un marco explícito GQM + PRAGMATIC para diseñar, evaluar y gobernar indicadores de ciberseguridad de IA.

Incluye:

- Un **marco integrativo GQM + PRAGMATIC** (a) que define principios, componentes y plantilla.
- Una **matriz de evaluación PRAGMATIC** (b) con puntuaciones sugeridas para las métricas clave.
- Una **narrativa explicativa** (c) para acompañar el diálogo con la alta dirección.
- Un **mapeo GQM de objetivos, preguntas y métricas** (d).
- Una **plantilla de Excel** (e) para articular objetivos, métricas, IGM y ROI.
- Una **plantilla de reporte ejecutivo** en PowerPoint (f).

## 2. Cómo usar este kit junto con la encuesta

1. Utilice la encuesta ISO/IEC 42001 para obtener una visión de madurez y prácticas actuales.
2. A partir de los resultados, seleccione las métricas más relevantes para su organización dentro del catálogo GQM.
3. Revise la matriz PRAGMATIC para valorar la calidad de cada métrica y descarte aquellas que no aporten suficiente valor.
4. Configure el libro Excel según la plantilla, vinculando métricas con objetivos y subíndices.
5. Elabore el reporte ejecutivo siguiendo la estructura propuesta, de forma que cada indicador se vea respaldado por un objetivo y una narrativa clara.

## 3. Buenas prácticas de gobierno de métricas

- **Trazabilidad**: ninguna métrica sin objetivo, ninguna diapositiva sin métrica.
- **Simplicidad**: pocas métricas estratégicas, bien entendidas y mantenidas; el resto, para uso operativo.
- **Revisión periódica**: actualice objetivos, preguntas y métricas al menos una vez al año.
- **Transparencia**: documente definiciones, fuentes, supuestos y limitaciones de cada métrica.

## 4. Adaptación al contexto nacional y sectorial

Los ejemplos están pensados para un contexto nacional similar al español, con marcos como ENS, ENIA y EU AI Act. Cada país o sector puede:

- Añadir objetivos específicos (por ejemplo, defensa, salud, transporte).
- Incorporar métricas regulatorias obligatorias.
- Ajustar pesos de subíndices e interpretación del IGM.

## 5. Limitaciones y sentido común

Ni GQM ni PRAGMATIC son talismanes infalibles. Pueden ayudar a evitar métricas inútiles y a reforzar las útiles, pero no sustituyen el juicio experto ni la responsabilidad de quienes toman decisiones. Si una métrica contradice la realidad manifiesta o induce comportamientos indeseados, el problema no está en la fórmula de Excel, sino en cómo se usa.

El mejor síntoma de que el kit funciona no es tener dashboards más bonitos, sino ver cambios reales en prácticas de seguridad, reducción de incidentes y conversaciones más maduras sobre la IA que queremos desplegar.
