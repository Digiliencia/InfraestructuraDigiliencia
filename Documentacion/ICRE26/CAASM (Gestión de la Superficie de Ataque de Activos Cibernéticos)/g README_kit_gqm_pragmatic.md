# README – Kit GQM + PRAGMATIC para Indicadores CAASM

Este kit complementa el “Kit de Encuesta CAASM” anterior con una capa adicional de rigor metodológico:

- Integra GQM (Goal–Question–Metric) para vincular objetivos nacionales con métricas.  
- Aplica PRAGMATIC para evaluar la calidad de cada métrica.

---

## Archivos incluidos

1. `marco_gqm_pragmatic.md`  
   - Explica el marco integrativo y proporciona ejemplos de cadenas GQM para los objetivos G1–G5.

2. `matriz_pragmatic_completa.md`  
   - Plantilla de evaluación PRAGMATIC para las métricas clave (M1.x–M5.x).

3. `narrativa_gqm_pragmatic.md`  
   - Narrativa explicativa para presentar el enfoque a comités técnicos y ejecutivos.

4. `mapeo_normativo_gqm_caasm.md`  
   - Tabla que relaciona cada métrica con requisitos normativos relevantes.

5. `plantilla_excel_gqm_pragmatic.md`  
   - Especificación de plantilla Excel para:  
     - Registrar valores de métricas.  
     - Calcular índices e IGM.  
     - Registrar puntuaciones PRAGMATIC.

6. `plantilla_reporte_ejecutivo_gqm_pragmatic.md`  
   - Guía para construir un reporte ejecutivo centrado en GQM + PRAGMATIC.

---

## Cómo usar este kit

1. **Definir objetivos nacionales (G)**  
   - Validar que G1–G5 representan los objetivos estratégicos deseados.  
   - Ajustar o ampliar según política nacional.

2. **Confirmar preguntas (Q)**  
   - Verificar que las Questions asociadas a cada Goal cubren los aspectos clave.

3. **Seleccionar métricas (M)**  
   - Partir de la lista propuesta M1.1–M5.2.  
   - Añadir o retirar métricas según contexto nacional.

4. **Evaluar con PRAGMATIC**  
   - Puntuar cada métrica con la matriz propuesta.  
   - Identificar métricas con baja relevancia o accionabilidad.

5. **Construir cuadro de mando nacional**  
   - Seleccionar un subconjunto manejable de métricas “estrella”.  
   - Asegurar trazabilidad GQM documentada.

6. **Iterar periódicamente**  
   - Revisar objetivos, preguntas y métricas cada 2–3 años.

---

## Relación con el Kit de Encuesta CAASM

- La encuesta recoge percepciones y prácticas organizativas.  
- El marco GQM + PRAGMATIC se aplica a indicadores agregados a partir de instrumentos como la encuesta y datos técnicos (CAASM, ASM, SIEM, etc.).

Ambos kits son complementarios: uno mide “cómo se ven las organizaciones”, el otro asegura que “lo que medimos a nivel país tiene sentido”.
