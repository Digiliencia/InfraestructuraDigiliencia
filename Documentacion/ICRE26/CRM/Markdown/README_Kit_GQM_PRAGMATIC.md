## **📚 CONCEPTOS CLAVE**

### **GQM: Trazabilidad Jerárquica**

```
GOAL (Estratégico)
  ↓ "¿Cómo lo logramos?"
QUESTION (Operacional)
  ↓ "¿Cómo lo medimos?"
METRIC (Técnico)
```

**Ejemplo:**
```
GOAL G4: Detectar incidentes rápidamente (National objective)
  ↓
QUESTION: ¿MTTD actual vs target <30 min?
  ↓
METRIC: MTTD = log_detection_timestamp - incident_creation_timestamp (minutes)
```

---

### **PRAGMATIC: 9 Criterios Viabilidad**

| Criterio | Pregunta | Ejemplo |
|---|---|---|
| **P — Predictivo** | ¿Predice breach o outcome? | MTTD ↓ = damage ↓ |
| **R — Relevante** | ¿Importa a stakeholders? | DORA requiere MTTD <30 min |
| **A — Accionable** | ¿Podemos actuar basados en datos? | IF MTTD >1h THEN escalate |
| **G — Genuino** | ¿Mide lo que pretende? | Timestamp objective; no gaming |
| **M — Significativo** | ¿Impacto material si mejora? | Cada 10 min = €500K saved |
| **A — Preciso** | ¿Operacionalmente definido? | Definición clara, auditable |
| **T — Oportuno** | ¿Disponible en tiempo relevante? | Real-time dashboard possible |
| **I — Independiente** | ¿Standalone o redundante? | Correlación aceptable MTTR |
| **C — Rentable** | ¿Costo colección justificado? | <€5K SIEM setup |

**Score PRAGMATIC = Promedio 9 criterios (0-5.0)**

## **❓ FAQ**

**P: ¿Diferencia GQM vs PRAGMATIC?**
A: GQM = qué medir (estrategia). PRAGMATIC = si es viable medirlo (operación).

**P: ¿Cuándo un métrica falla PRAGMATIC?**
A: Si <3.0/5.0 score. Ej: "Satisfacción CISO" = 1.5/5 (subjetivo, gaming-prone).

**P: ¿Reemplaza GQM + PRAGMATIC otras frameworks?**
A: No; complementa NIST CSF 2.0, ISO 27001. Operacionaliza sus requisitos.

**P: ¿Cómo uso con auditor externo?**
A: Mapeo Normativo (c) demuestra trazabilidad GDPR/DORA/NIS2 → auditor lo valida rápido.

**P: ¿Frecuencia update indicadores?**
A: MTTD/MTTR daily; Patch % weekly; Compliance annual; IGM annual.

**P: ¿Dónde sourcing data?**
A: SIEM (MTTD), Tickets (MTTR), Vuln scanner (Patch %), Phishing platform (Click), etc.

**P: ¿Cost implementation?**
A: Kit = gratis. Excel setup 1-2 horas. SIEM-ready orgs ~€50K; nuevas SIEM €300K+.

**P: ¿Timeline from 0 → full implementation?**
A: 12 weeks baseline; 24 weeks full optimization.
