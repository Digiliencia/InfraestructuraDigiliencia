├─ Input scores en Excel (e) Plantilla IGM/ROI
├─ Excel auto-calcula: IGM, dominios, gaps
├─ Crear matriz gaps impacto×esfuerzo
└─ Definir top 5-7 iniciativas prioritarias

SEMANA 5: Reporte
├─ Crear PowerPoint usando (f) Template 12-slides
├─ Customizar con data vuestra organización
├─ Validar con junta
└─ Aprobación roadmap + presupuesto
