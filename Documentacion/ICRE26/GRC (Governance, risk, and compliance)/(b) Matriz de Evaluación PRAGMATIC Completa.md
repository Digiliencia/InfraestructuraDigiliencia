# Matriz de Evaluación PRAGMATIC Completa para indicadores GRC nacionales

Versión 1.0 – 2026

## 1. Introducción

Esta matriz aplica los nueve criterios PRAGMATIC a un subconjunto representativo de indicadores GRC nacionales de ciberseguridad definidos en el marco GQM. La escala es de 0 a 3:

- 0 = No cumple.  
- 1 = Cumple pobremente.  
- 2 = Cumple razonablemente.  
- 3 = Cumple de forma sólida.

Las valoraciones son orientativas y deben revisarse en ejercicios posteriores según la experiencia acumulada.

## 2. Leyenda de criterios

- **PRED:** Predictivo.  
- **REL:** Relevante.  
- **ACC:** Accionable.  
- **GEN:** Genuino.  
- **SIG:** Significativo.  
- **PRE:** Preciso.  
- **OPOR:** Oportuno.  
- **IND:** Independiente.  
- **REN:** Rentable.

## 4. Uso de la matriz

- Para cada nueva métrica propuesta, cumplimentar una fila similar.  
- Revisar periódicamente las puntuaciones PRAGMATIC según experiencia.  
- Detectar métricas con baja puntuación global (p. ej. < 18/27) para reformulación o retiro.

Con ello se evita que el sistema nacional de indicadores se convierta en una colección de números decorativos sin valor operativo.
