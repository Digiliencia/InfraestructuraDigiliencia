# README – Kit de Encuesta TLPT / DORA / TIBER-ES

## Contenido del kit

El kit incluye los siguientes componentes (en formato Markdown y listos para adaptar):

1. `01-encuesta-TLPT.md`: Modelo de encuesta integral para responsables de seguridad, riesgos y negocio.  
2. `02-guia-metodologica-TLPT-encuesta.md`: Guía para interpretar y puntuar la encuesta (IGM).  
3. `03-narrativa-explicativa-TLPT-encuesta.md`: Narrativa que contextualiza el uso de la encuesta.  
4. `04-mapeo-preguntas-normativa-TLPT.md`: Tabla que vincula preguntas con DORA, RTS TLPT y TIBER-ES.[web:16][web:38]  
5. `05-plantilla-excel-IGM-ROI-TLPT.md`: Instrucciones para construir una plantilla Excel de cálculo de IGM y ROI.  
6. `06-plantilla-reporte-ejecutivo-TLPT-PPT.md`: Esqueleto de reporte ejecutivo para comités.  
7. `07-README-kit-encuesta-TLPT.md`: Este documento de orientación.

## Uso recomendado

1. Adaptar la encuesta a la realidad de la entidad (terminología, ámbitos específicos).  
2. Definir el método de recogida (formularios online, entrevistas guiadas, talleres).  
3. Poner en marcha la plantilla Excel para poder calcular IGM y ROI con datos reales.  
4. Utilizar el reporte ejecutivo para presentar resultados en órganos de gobierno y ante supervisores.

## Requisitos normativos cubiertos

El kit está alineado con:

- DORA (Reglamento (UE) 2022/2554) en sus pilares de gestión de riesgos TIC, pruebas de resiliencia, gestión de terceros TIC e incidentes.[web:26]  
- RTS conjuntos de las ESAs sobre TLPT (JC 2024-29), que concretan criterios de selección de entidades, enfoque metodológico y requerimientos de attestation.[web:16][web:31]  
- Marco TIBER-EU y su implementación española TIBER-ES, que definen las fases TTI, Red Team y cierre a nivel operativo.[web:38]  

## Licencia y personalización

- El kit se ofrece como base para su adaptación por entidades, asociaciones sectoriales y supervisores.  
- Se anima a enriquecerlo con métricas internas adicionales, siempre que se mantenga la trazabilidad normativa.  
- La ironía utilizada en las opciones de respuesta puede modularse si se requiere una versión “para público muy serio”.

## Contacto y mejoras

- Se recomienda mantener un registro de cambios internos de la organización al adaptar el kit.  
- Cualquier mejora basada en lecciones aprendidas de TLPT reales debería integrarse en futuras versiones del cuestionario y de las plantillas de análisis.
