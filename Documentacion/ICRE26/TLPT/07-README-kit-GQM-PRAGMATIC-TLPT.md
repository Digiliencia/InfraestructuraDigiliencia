# README – Kit GQM + PRAGMATIC para indicadores TLPT

## 1. Qué incluye este kit

1. `01-marco-GQM-PRAGMATIC-TLPT.md`  
   - Define objetivos G1–G5, preguntas GQM y métricas M1–M16.  
2. `02-matriz-evaluacion-PRAGMATIC-TLPT.md`  
   - Matriz PRAGMATIC con valoración orientativa de cada métrica.  
3. `03-narrativa-explicativa-GQM-PRAGMATIC-TLPT.md`  
   - Explica el sentido del marco, su conexión con DORA/TIBER‑ES y cómo usarlo.  
4. `04-mapeo-GQM-PRAGMATIC-normativa-TLPT.md`  
   - Tabla que vincula cada métrica con artículos/secciones relevantes de DORA, RTS TLPT y TIBER‑ES.[web:16][web:38]  
5. `05-plantilla-excel-IGM-ROI-GQM-PRAGMATIC.md`  
   - Guía para construir en Excel el IGM global y el ROI TLPT.  
6. `06-reporte-ejecutivo-TLPT-GQM-PRAGMATIC-PPT.md`  
   - Esqueleto de presentación ejecutiva para comités y supervisores.  
7. `07-README-kit-GQM-PRAGMATIC-TLPT.md`  
   - Este documento.

## 2. Público objetivo

- Equipos de ciberseguridad, riesgos y continuidad de entidades sujetas a DORA.  
- Órganos de supervisión que quieran armonizar métricas TLPT a nivel nacional/sectorial.[web:16][web:26]  
- Asociaciones sectoriales que deseen construir benchmarks de resiliencia.  

## 3. Uso recomendado (ruta rápida)

1. Seleccionar subconjunto de métricas M1–M16 que la entidad pueda medir con calidad aceptable (PRAGMATIC ≥ 2 de media).  
2. Implementar la hoja Excel IGM/ROI y alimentar los valores tras cada TLPT.  
3. Usar el reporte ejecutivo para “traducir” resultados técnicos a lenguaje de negocio.  
4. Revisar anualmente las métricas (alta/baja) según su utilidad real y coste de obtención.

## 4. Alineamiento normativo

El kit está diseñado para ser compatible con:

- DORA (Reglamento (UE) 2022/2554) – artículos sobre gobierno TIC, pruebas de resiliencia, gestión de terceros e incidentes.[web:26]  
- RTS conjuntos de las ESAs sobre TLPT (JC 2024‑29) – criterios de selección, metodología, duración, cierre, attestation.[web:16]  
- TIBER‑EU y TIBER‑ES – guías operativas para TTI, Red Team, cierre y remediación.[web:38]

No reemplaza ningún requisito oficial, pero ayuda a estructurar la conversación entre entidades y autoridades.

## 5. Licencia interna y estilo

- Puede adaptarse libremente dentro de la organización o a nivel sectorial, manteniendo la trazabilidad a los marcos normativos mencionados.  
- Las descripciones con tono irónico son opcionales; si hay riesgo de choque cultural, siempre cabe ofrecer una versión “modo notario”.
