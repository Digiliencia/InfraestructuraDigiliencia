# README – Kit de Encuesta CMMC con Marco GQM + PRAGMATIC

> Versión: 1.0

---

## 1. Qué incluye este kit

- `marco_integrativo_gqm_pragmatic.md` – Explica el enfoque GQM aplicado a indicadores CMMC/NIS2/ENS y la evaluación PRAGMATIC.  
- `matriz_evaluacion_pragmatic_completa.md` – Tabla con las principales métricas y su puntuación PRAGMATIC.  
- `narrativa_explicativa_gqm_pragmatic.md` – Texto narrativo para explicar el porqué del enfoque a públicos no técnicos.  
- `mapeo_preguntas_normativa_gqm_pragmatic.md` – Tabla que conecta cada pregunta de la encuesta con su objetivo GQM y los requisitos normativos.  
- `plantilla_excel_igm_roi_gqm_pragmatic.md` – Guía para construir el Excel de IGM y ROI incorporando GQM y PRAGMATIC.  
- `plantilla_reporte_ejecutivo_ppt_gqm_pragmatic.md` – Esqueleto de presentación ejecutiva incluyendo la dimensión de calidad de métricas.  

---

## 2. Cómo usar este kit junto a la encuesta

1. **Diseñar la encuesta** usando el modelo de preguntas (kit anterior) y el mapeo GQM de este kit.  
2. **Recoger respuestas** y volcarlas en el Excel definido en `plantilla_excel_igm_roi_gqm_pragmatic.md`.  
3. **Calcular métricas**: MFA, vulnerabilidades, MTTD/MTTR, cifrado, proveedores, formación, etc.  
4. **Aplicar PRAGMATIC**: revisar o ajustar las puntuaciones por métrica según la realidad de tu organización.  
5. **Construir el IGM** y los índices parciales usando sólo métricas de calidad aceptable.  
6. **Preparar el reporte ejecutivo** con la plantilla PowerPoint, explicando no sólo el “qué” sino también el “cómo” y el “por qué” de las métricas.

---

## 3. Escenarios de uso recomendados

- **Organizaciones individuales** que quieren madurar su cuadro de mando de ciberseguridad.  
- **Grupos empresariales** que necesitan comparar unidades de negocio con un lenguaje común.  
- **Iniciativas sectoriales o nacionales** que quieren construir índices agregados basados en indicadores robustos.  

---

## 4. Consejos prácticos

- No intentes usar todas las métricas posibles desde el primer día: empieza por un subconjunto de alto impacto y alta puntuación PRAGMATIC.  
- Revisa periódicamente las puntuaciones PRAGMATIC: una métrica puede ganar o perder relevancia con el tiempo.  
- Documenta las decisiones: por qué se incluye/excluye una métrica, qué umbrales se consideran aceptables, etc.  

---

## 5. Limitaciones

- La calibración de PRAGMATIC implica un componente de juicio experto; no es “verdad revelada”.  
- Los objetivos GQM deben revisarse si cambian la estrategia o el marco regulatorio (por ejemplo, nuevas obligaciones de NIS2).  
- Este kit no sustituye marcos formales de certificación, pero puede hacerlos más eficientes y menos traumáticos.
