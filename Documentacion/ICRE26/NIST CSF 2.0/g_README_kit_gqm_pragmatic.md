# README – Kit GQM + PRAGMATIC para Indicadores NIST CSF 2.0

## 1. Qué incluye este kit

- Marco conceptual integrador GQM + PRAGMATIC.  
- Matriz de evaluación PRAGMATIC para métricas clave.  
- Narrativa explicativa para informes y presentaciones.  
- Mapeo de métricas a NIST CSF 2.0 y marcos normativos.  
- Especificación de plantilla Excel para IGM y ROI.  
- Guion para reporte ejecutivo en PowerPoint.

## 2. Cómo usarlo

1. Revisar el marco GQM + PRAGMATIC y seleccionar los objetivos estratégicos relevantes.  
2. Ajustar el conjunto de métricas a la realidad organizativa o nacional.  
3. Puntuar las métricas según los criterios PRAGMATIC.  
4. Implementar la hoja Excel para calcular IGM, PRAGMATIC y ROI.  
5. Presentar resultados con la plantilla de reporte ejecutivo.

## 3. Adaptación

- El conjunto de métricas y las puntuaciones PRAGMATIC son orientativos.  
- Se recomienda involucrar a representantes de negocio, riesgo, tecnología y cumplimiento en la elección final de métricas.  

## 4. Objetivo final

Disponer de un sistema de medición de ciberseguridad que:

- Esté alineado con NIST CSF 2.0 y los objetivos de resiliencia.[web:5][web:22]  
- Ofrezca trazabilidad desde el objetivo estratégico hasta el dato técnico mediante GQM.[web:35][web:41][web:46]  
- Utilice métricas que realmente aporten valor a la toma de decisiones, evaluadas bajo PRAGMATIC.[web:40][web:44][web:47]
