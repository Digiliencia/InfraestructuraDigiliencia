# Mapeo de indicadores GQM a requisitos normativos (alto nivel)

| Indicador / Métrica | Función CSF 2.0 | Ejemplos NIST CSF 2.0 relevantes | Normativa / marcos relacionados (alto nivel) |
| --- | --- | --- | --- |
| % cuentas alto riesgo con MFA | PR | PR.AA: gestión de identidades y autenticación fuerte | ENS (control de acceso); ISO/IEC 27001 A.5, A.8; NIS2 (gestión de acceso) |
| Variación cobertura MFA | PR | PR.AA | ISO/IEC 27004 (medición de seguridad); marcos GRC |
| MTTD incidentes graves (horas) | DE | DE.CM: monitoreo continuo; DE.DP: detección de eventos | ENS (detección de intrusiones); ISO/IEC 27035; NIS/NIS2 (detección y notificación) |
| Tendencia MTTD (%) | DE | DE.CM, DE.DP | ISO/IEC 27004; guías de métricas de incidentes |
| % servicios críticos que cumplen RTO | RC | RC.RP: planes de recuperación; RC.IM: pruebas | ISO 22301 (continuidad); ENS (continuidad del servicio); DORA (resiliencia operativa digital) |
| Evolución % cumplimiento RTO | RC | RC.RP, RC.IM | ISO 22301; marcos sectoriales de continuidad |
| % activos detectados automáticamente | ID | ID.AM: inventario de activos | ENS (inventario de activos); ISO/IEC 27001 A.5, A.8 |

| % proveedores críticos con cláusulas de ciberseguridad | GV.SC | GV.SC: gestión de riesgo de cadena de suministro | NIS2 (cadena de suministro); DORA; ISO/IEC 27036 |
