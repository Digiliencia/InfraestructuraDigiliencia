## ¿QUÉ ES ESTE KIT?

Este kit proporciona un **marco de clase mundial** para diseñar, evaluar e implementar indicadores de ciberseguridad basados en EPSS, con garantías de:

✓ **Trazabilidad GQM:** Cada métrica remonta a objetivo estratégico  
✓ **Practicidad PRAGMATIC:** 9 criterios de viabilidad operacional  
✓ **Cumplimiento Regulatorio:** Mapeo a NIS2, GDPR, ENS, ISO 27001  
✓ **Justificación Financiera:** ROI/ROSI cuantificado (430% en ejemplo)

**Diferencial:** A diferencia de "listas de métricas genéricas", este kit responde:
- ¿CUÁL es la métrica? (definición matemática)
- ¿POR QUÉ? (objetivo estratégico GQM)
- ¿CÓMO se mide? (formula, frecuencia, data sources)
- ¿ES PRÁCTICA? (9 criterios PRAGMATIC: puntuación 7.8/10 promedio)
- ¿VALE LA PENA? (ROI 430%)

## INTERPRETACIÓN DE RESULTADOS

### PRAGMATIC Score

```
9.0-10.0: ⭐ EXCELENTE
└─ Métrica lista para producción sin cambios

8.0-8.9: ⭐⭐ MUY BUENA
└─ Implementar con documentación clara

7.0-7.9: ⭐⭐⭐ BUENA
└─ Implementar pero mejorar (precisión, automatización)

6.0-6.9: ⭐⭐⭐⭐ ACEPTABLE
└─ Reconsiderar o mejorar significativamente

<6.0: ⭐⭐⭐⭐⭐ DEFICIENTE
└─ NO RECOMENDAR implementación hasta mejora
```

### IGM Global

```
IGM = PROMEDIO(PRAGMATIC scores todas métricas)

8.5-10.0: EXCELENTE (programa maduro, auditable)
7.5-8.4: BUENA (programa operativo, con mejoras posibles)
6.5-7.4: ACEPTABLE (programa básico, requiere inversión)
<6.5: DÉBIL (riesgo alto de error en decisiones)
```

## PREGUNTAS FRECUENTES (FAQ)

**P: ¿Puedo usar este kit si NO estoy en España?**  
R: Sí. Reemplaze "NIS2/GDPR/ENS" con regulación local (CCPA, HIPAA, etc). 
Framework GQM+PRAGMATIC es agnóstico.

**P: ¿Cuál es la diferencia vs "top 30 metrics lists"?**  
R: Trazabilidad. Listas genéricas dicen "mida MTTR". 
Este kit dice: "MTTR responde pregunta Q1.1, que sirve objetivo OB-1, 
que mapea a NIS2 Art 21(E), con practicidad PRAGMATIC 7.6/10, 
ROI 430%, ejemplo actual 14 días vs target <7 días."

**P: ¿Necesito Excel específicamente?**  
R: No. Google Sheets, Tableau, Power BI, cualquier tool funciona. 
Importa la estructura, no el software.

**P: ¿Cada cuánto recalcular IGM?**  
R: Recomendado: Anualmente (cambios significativos).
Trimestralmente: Para tracking de KPIs.
Mensualmente: Para métricas operacionales como MTTR.

**P: ¿Puedo ignorar métricas con PRAGMATIC <7.0?**  
R: Sí, si costo > beneficio. Pero documente por qué se omite.
Recomendación: Implementar todo ≥7.0.
