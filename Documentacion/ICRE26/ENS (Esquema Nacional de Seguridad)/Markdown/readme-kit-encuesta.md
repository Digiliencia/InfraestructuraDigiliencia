## ❓ FAQ - 10 PREGUNTAS COMUNES

### **P1: ¿Cuánto tiempo toma responder la encuesta?**
**R:** 45-90 minutos por respondente. 114 preguntas × escala 1-5.
Recomendación: Responder en bloques (no de golpe).

---

### **P2: ¿Quiénes deben responder?**
**R:** 5 respondentes idealmente:
- 1 CISO/Responsable Seguridad (40% peso)
- 1 Cumplimiento (20% peso)
- 1 Auditoría (20% peso)
- 1 Ejecutivo negocio (10% peso)
- 1 RRHH (10% peso)

Alternativa: Si <5 personas, ponderar según disponibilidad.

---

### **P3: ¿Qué significa IGM?**
**R:** Índice de Gestión de Madurez (1.0-5.0).
- 1.0-1.9 = CRÍTICO (riesgo extremo)
- 2.0-2.9 = INSUFICIENTE (riesgo alto)
- 3.0-3.4 = BÁSICO (riesgo medio)
- 3.5-4.2 = AVANZADO (riesgo bajo)
- 4.3-5.0 = EXCELENCIA (mínimo riesgo)

---

### **P4: ¿Cómo se calcula el IGM?**
**R:** Promedio ponderado de 6 dominios:
```
IGM = (MO×0,20 + MOP×0,25 + MP×0,20 + 
       Inc×0,15 + Cap×0,10 + Gob×0,10) / 100
```
Donde:
- MO = Marco Organizativo
- MOP = Marco Operacional
- MP = Medidas Protección
- Inc = Incidentes
- Cap = Capacitación
- Gob = Gobernanza

---

### **P5: ¿Qué pasa si respondentes dan respuestas muy diferentes?**
**R:** Es NORMAL. Refleja perspectivas distintas.
Acción:
- Analizar desviación (SD > 1.5 = debate necesario)
- Hacer reunión con respondentes para alinearse
- Segunda ronda de respuestas con mejor contexto
- Usar media ponderada (promedio final)

---

### **P6: ¿Cuál es el ROI esperado?**
**R:** Depende reducción incidentes:
```
Escenario Base (40% reducción):
- Año 1: -12% (inversión fuerte = -100K€, beneficio +198K€)
- Año 2+: +150K€/año sostenible
- Payback: 11 meses
- NPV 3 años: 349K€
```

---

### **P7: ¿Necesitamos auditoría externa?**
**R:** Recomendación:
- PRIMERA VEZ: Sí (validar diagnóstico)
- Costo: 15-30K€
- Plazo: 3-4 semanas
- Resultado: Gap analysis + remediación

---

1. ✓ Este README (inicio)
2. ✓ Encuesta (114 preguntas)
3. ✓ Excel IGM (cálculos)

Complementarios (no esencial):

---

### **P9: ¿Podemos adaptar la encuesta?**
**R:** Sí, recomendaciones:
- ✓ Añadir 5-10 preguntas propias
- ✓ Modificar textos (para claridad)
- ✗ No eliminar preguntas (rompe análisis)
- ✗ No cambiar escala 1-5 (perderá comparabilidad)

---

### **P10: ¿Cómo mantenemos confidencialidad?**
**R:** Buenas prácticas:
- Encuesta anónima (no pedir nombre)
- HTTPS cifrado (Google Forms lo hace)
- Acceso restringido (solo CISO, CFO, Auditoría)
- No compartir resultados individuales
- GDPR cumplimiento (si datos personales)

## 📊 PASO A PASO: ANÁLISIS DESPUÉS ENCUESTA

### **A. EXPORTAR RESPUESTAS (5 min)**

```
Google Forms:
1. Click "Respuestas" tab
2. Click icono Sheets (∴)
3. Crear nueva hoja en Google Sheets
4. Automático: Copia todas respuestas
5. Descargar: File → Download → CSV o Excel
```

---

### **B. CONSOLIDAR EN EXCEL (30 min)**

```
1. Abrir documento "plantilla-excel-igm-roi.md"
2. Crear Hoja 1 "Consolidación"
3. Copiar estructura:
   - Col A: P1-P114 (números pregunta)
   - Col B: Sección (I-XI)
   - Col C: Tema
   - Col D-H: Respuestas CISO, Cumpl, Aud, Ejec, RRHH
   - Col I: Media ponderada (fórmula automática)
4. Validar rango 1-5 (fórmula de error)
```

---

### **C. CALCULAR IGM (15 min)**

```
1. Crear Hoja 2 "IGM"
2. Calcular promedio por dominio:
   - Dominio 1 (MO): =PROMEDIO(I6:I16)
   - Dominio 2 (MOP): =PROMEDIO(I17:I35)
   - ... etc 6 dominios
3. Fórmula IGM global: [ver documento Excel]
4. Resultado: Número 1.0-5.0
5. Interpretación automática (CRÍTICO/EXCELENCIA)
```

---

### **D. ANÁLISIS ROI (30 min)**

```
1. Crear Hoja 3 "ROI"
2. Inputs:
   - Gasto actual: P105 respuesta
   - Inversión recomendada: 75K€ (default)
   - Costo incidente: P107
   - Incidentes/año: P51
   - Reducción esperada: 40% (default)
3. Fórmulas automáticas:
   - Beneficio evitados: incidentes × costo × reducción
   - ROI: (beneficios - inversión) / inversión × 100
   - Payback: inversión / beneficio anual
   - NPV 3 años: cálculo VAN
4. Resultado: ROI %, Payback meses, NPV €
```

---

### **E. DASHBOARD EJECUTIVO (20 min)**

```
1. Crear Hoja 4 "Dashboard"
2. KPIs principales:
   - IGM actual (Hoja 2)
   - Brecha a meta (4.0 - IGM actual)
   - MTTD, MTTR, Incidentes (respuestas P47, P48, P51)
   - Conformidad ENS/NIS2 (P95, P97)
   - Presupuesto (P105, P106)
3. Gráficos:
   - Radar (madurez por dominio)
   - Line (histórica + proyección)
   - Waterfall (ROI)
   - Heat Map (estado por dominio)
4. Alertas automáticas (IF statements)
```

---
