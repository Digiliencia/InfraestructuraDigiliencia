# README – Kit GQM + PRAGMATIC para métricas de Agentic IA

## 1. Contenido

El kit incluye los siguientes archivos Markdown (.md):

1. `A-Marco-GQM-PRAGMATIC.md` – Marco integrativo conceptual.  
2. `B-Matriz-PRAGMATIC-Completa.md` – Tabla de evaluación PRAGMATIC por métrica.  
3. `C-Narrativa-GQM-PRAGMATIC.md` – Narrativa explicativa para decisores.  
4. `D-Mapeo-Preguntas-Normativa-GQM.md` – Mapeo de objetivos/preguntas a marcos normativos.  
5. `E-Plantilla-Excel-GQM-PRAGMATIC.md` – Especificación de plantilla de Excel para GQM + PRAGMATIC, IGM Agentic y ROI.  
6. `F-Reporte-Ejecutivo-GQM-PRAGMATIC.md` – Guion de presentación ejecutiva.  

## 2. Objetivo

Proporcionar un conjunto coherente de materiales para:

- Derivar métricas de Agentic IA a partir de objetivos nacionales y organizativos mediante GQM.  
- Evaluar la utilidad de cada métrica usando los criterios PRAGMATIC.  
- Integrar los resultados en índices de madurez (IGM Agentic) y análisis de ROI.

## 3. Uso recomendado

1. Leer el marco (`A-Marco-GQM-PRAGMATIC.md`).  
2. Ajustar la matriz PRAGMATIC a la realidad de la organización (`B-Matriz-PRAGMATIC-Completa.md`).  
3. Configurar la plantilla Excel según `E-Plantilla-Excel-GQM-PRAGMATIC.md`.  
4. Generar un reporte ejecutivo (`F-Reporte-Ejecutivo-GQM-PRAGMATIC.md`).  

## 4. Actualización

Dado que las métricas y los marcos de riesgo de Agentic IA evolucionan rápidamente, se recomienda revisar y actualizar el kit al menos anualmente.
