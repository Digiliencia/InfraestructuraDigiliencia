# README – Kit de Encuesta sobre Agentic IA y Ciberseguridad

## 1. Contenido del kit

Este kit incluye los siguientes archivos en formato Markdown (.md):

1. `00-README-Kit-Encuesta.md` – Este documento.  
2. `01-Encuesta-Agentic-AI.md` – Modelo de encuesta integral para organizaciones.  
3. `02-Guia-Metodologica.md` – Guía para aplicar, puntuar e interpretar la encuesta.  
4. `03-Narrativa-Explicativa.md` – Texto explicativo para decisores y equipos.  
5. `04-Mapeo-Preguntas-Normativa.md` – Tabla de mapeo de preguntas a RGPD, AEPD, NIST, ENISA y perfil Agentic IA.  
6. `05-Plantilla-Excel-IGM-ROI.md` – Especificación para crear la hoja de cálculo de IGM y ROI.  
7. `06-Reporte-Ejecutivo-PPT.md` – Guion para preparar una presentación ejecutiva.

## 2. Objetivo

El kit permite:

- Evaluar el grado de madurez de una organización frente a la Agentic IA.  
- Medir la adopción de métricas como UAR, PAR, PED, RRS, CES y TTC.  
- Verificar alineamiento con RGPD, guía AEPD, NIST AI RMF, perfil Agentic IA y marcos de ENISA.  
- Estimar un ROI que incorpore no solo productividad, sino también resiliencia y reducción de incidentes.

## 3. Cómo usarlo (resumen rápido)

1. Leer `03-Narrativa-Explicativa.md` para entender el enfoque.  
2. Aplicar la encuesta (`01-Encuesta-Agentic-AI.md`) con los responsables clave.  
3. Utilizar la `02-Guia-Metodologica.md` para puntuar respuestas y obtener el IGM.  
4. Construir la hoja de cálculo de acuerdo con `05-Plantilla-Excel-IGM-ROI.md`.  
5. Preparar el reporte ejecutivo siguiendo `06-Reporte-Ejecutivo-PPT.md`.  
6. Consultar `04-Mapeo-Preguntas-Normativa.md` para revisar cumplimiento normativo.

## 4. Actualización y mantenimiento

Dado que el perfil de Agentic IA y las guías regulatorias están evolucionando rápidamente, se recomienda:

- Revisar y actualizar el kit al menos una vez al año.  
- Incorporar nuevas métricas y requisitos normativos que surjan de ENISA, AEPD, NIST u otros organismos.

## 5. Licencia y adaptación

El contenido está diseñado para ser adaptado por la organización:

- Puede personalizarse el lenguaje, las escalas y las ponderaciones.  
- Se recomienda mantener la trazabilidad con las fuentes (CLTC Berkeley, AEPD, ENISA) al realizar adaptaciones.
