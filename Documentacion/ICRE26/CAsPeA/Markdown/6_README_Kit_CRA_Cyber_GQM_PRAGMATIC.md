## ¿QUÉ ES CRA-CYBER GQM+PRAGMATIC?

Es un **kit integral de evaluación de madurez ciberseguridad** que:

✓ Alinea **objetivos estratégicos nacionales** con **métricas técnicas operacionales**  
✓ Valida que cada métrica sea **prácticamente útil** (PRAGMATIC)  
✓ Proporciona **trazabilidad completa** desde regulación hasta datos técnicos  
✓ Cuantifica **costes reales de personal** (CAsPeA)  
✓ Demuestra **ROI defensible** ante CFO y Junta Directiva  

---

## NOVEDAD: INTEGRACIÓN GQM+PRAGMATIC

**GQM (Goal-Question-Metric, 1985):**
- Metodología top-down de alineación de objetivos
- Establece cadena: Objetivo → Preguntas → Métricas medibles
- Usado por NASA, Motorola, Microsoft

**PRAGMATIC (Glasgow et al., 2013):**
- 9 criterios para validar *utilidad práctica* de métricas
- Predictivo, Relevante, Accionable, Genuino, Significativo, Preciso, Oportuno, Independiente, Rentable

**Integración única en marco español:**
- GQM define *QUÉ medir* (alineación estratégica)
- PRAGMATIC valida *SI es útil* (calidad métrica)
- Resultado: Métrica excelente = Estratégica + Útil + Implementable

```
├─ Preciso: 4/4 (±2% error)
├─ Oportuno: 4/4 (Mensual disponible)
├─ Independiente: 4/4 (No covarría)
├─ Rentable: 4/4 (Datos internos)
└─ MEDIA: 3.8/4 ✓ EXCELENTE → IMPLEMENTAR
```

**Para:** Analistas de datos, Auditores, CISOs técnicos  
**Uso:** Validación de calidad métrica + decisión GO/NO-GO

```
1.1) Descargar 6 documentos desde repositorio
1.2) Revisar Marco GQM+PRAGMATIC (Doc 1)
```

### PASO 2: LANZAMIENTO ENCUESTA (Semana 3)

```
2.1) Configurar Google Forms / Qualtrics
2.2) Copiar 58 preguntas desde Doc 1
2.3) Enviar invitaciones (150-250 respondientes)
```

### PASO 3: ANÁLISIS (Semana 6-8)

```
3.1) Exportar respuestas a Excel (Doc 4, Hoja 1)
3.2) Sistema valida automáticamente
```

```
5.1) Adaptar PowerPoint (Doc 5) con datos reales
5.2) Ensayar presentación (20-25 min)
```

## FAQ — PREGUNTAS FRECUENTES

**P: ¿Cuánto tiempo toma completar la encuesta?**  
R: 45-60 minutos en línea. Si es entrevista guiada, 90 minutos.

**P: ¿Cuántas personas debo encuestar?**  
R: 150-250 para dataset sólido. Mínimo 50 para piloto.

**P: ¿Qué pasa si IMG <40?**  
R: Crítico. Recomendación: Contratación consultor externo + evaluación profunda.

**P: ¿Cada cuánto debo repetir?**  
R: Anualmente obligatorio. Semestral recomendado post-implementación mejoras.

**P: ¿Los datos son confidenciales?**  
R: Sí. Anonimizados, cumplimiento GDPR, solo análisis agregado.

**P: ¿Cómo interpreto PRAGMATIC score <3.0?**  
R: Métrica poco útil. Revisar definición, complementar con métrica alternativa, o usar solo con auditor externo.
