# README – Kit GQM + PRAGMATIC + PASTA+STRIDE
## De los objetivos nacionales a las métricas útiles (y sólo a esas)

### 1. Contenido del kit

Este kit complementa el kit de encuesta PASTA+STRIDE y añade una capa de disciplina métrica:

1. `01-marco-integrativo-GQM-PRAGMATIC.md`  
   Marco conceptual que conecta objetivos (G), preguntas (Q) y métricas (M) y les aplica el filtro PRAGMATIC.[web:45][web:50][web:52][web:58]  

2. `02-matriz-evaluacion-PRAGMATIC-completa.md`  
   Tabla con evaluación 1–5 de cada métrica clave frente a los nueve criterios PRAGMATIC.

3. `03-narrativa-explicativa-GQM-PRAGMATIC.md`  
   Texto para explicar a decisores por qué este enfoque mejora el típico “PowerPoint de cifras”.  

4. `04-mapeo-GQM-metricas-requisitos-normativos.md`  
   Relación entre métricas y requisitos de NIS2, ENS, DORA, España Digital, etc.[web:21][web:24][web:27][web:30]  

5. `05-plantilla-Excel-IGM-ROI-GQM-PRAGMATIC.md`  
   Instrucciones para implementar en Excel el cálculo de un IGM y un ROI aproximado basados en las métricas seleccionadas.[web:48][web:51]  

6. `06-plantilla-reporte-ejecutivo-GQM-PRAGMATIC.md`  
   Esqueleto para presentar resultados a nivel ejecutivo o político.

### 2. Uso previsto

- **A nivel nacional o sectorial**:  
  – Definir y mantener un cuadro de mando coherente con objetivos de planes nacionales y europeos.  
  – Evaluar la efectividad de políticas y regulaciones en el tiempo.  

- **A nivel organizativo**:  
  – Descubrir qué métricas actuales sobran y cuáles faltan.  
  – Vincular prácticas de Threat Modeling con decisiones de inversión y priorización.  

- **A nivel proyecto**:  
  – Asegurar que lo que se mide en una aplicación crítica tiene sentido cuando se mira desde el regulador o el ciudadano.

### 3. Pasos recomendados

1. Revisar y, si es necesario, adaptar los objetivos G1–G4 a la realidad institucional.  
2. Seleccionar, a partir de la matriz PRAGMATIC, un subconjunto manejable de métricas núcleo.  
3. Implementar la plantilla de Excel para obtener IGM y estimaciones de ROI.  
4. Integrar estas métricas en los ciclos de reporting existentes (informes de estado, cuadros de mando, etc.).  
5. Revisar anualmente el portafolio de métricas: introducir nuevas, retirar las que han dejado de ser útiles.

### 4. Relación con el kit de encuesta PASTA+STRIDE

El kit de encuesta mide la madurez organizativa en modelado de amenazas y gestión de riesgos. Este kit va un paso más allá:

- Toma algunos indicadores estructurales (cobertura, simulaciones, tiempos de recuperación, integración en riesgo/ROI).  
- Los arma dentro de GQM.  
- Los somete al juicio PRAGMATIC para priorizar la recogida de datos más eficiente y significativa.[web:18][web:31][web:50][web:52][web:58]

Ambos kits se pueden usar juntos: la encuesta como instrumento de captura de datos; GQM+PRAGMATIC como brújula para decidir qué merece mantenerse como métrica estratégica.

### 5. Licencia y adaptabilidad

Está pensado para ser adaptado:

- Cambiando pesos PRAGMATIC y de IGM.  
- Introduciendo métricas adicionales específicas de sector (salud, energía, finanzas, etc.).  
- Afinando el mapeo a normativas locales o internas de la organización.

### 6. Epílogo sin métricas (pero con intención)

Si algo persigue este kit es evitar dos extremos igualmente peligrosos:

- El del “todo se mide” sin preguntarse para qué.  
- El del “nadie sabe qué medir” y, por tanto, todo se decide por intuición o presión del titular del día.

Entre ambos extremos, GQM y PRAGMATIC ofrecen un camino razonable: menos métricas, mejores métricas, y mucha más capacidad de explicar, con cierto aplomo, por qué se invierte donde se invierte y qué se espera obtener a cambio.
