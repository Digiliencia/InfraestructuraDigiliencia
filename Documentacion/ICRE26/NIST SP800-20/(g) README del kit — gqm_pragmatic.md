# Kit GQM + PRAGMATIC para Indicadores Nacionales de Ciberseguridad

Este kit recoge los materiales necesarios para diseñar, evaluar y comunicar un sistema de indicadores nacionales de ciberseguridad basado en GQM y evaluado con la matriz PRAGMATIC.

## Contenido del kit

1. `marco_gqm_pragmatic.md`: Marco Integrativo GQM + PRAGMATIC.
2. `matriz_pragmatic.md`: Matriz de Evaluación PRAGMATIC Completa.
3. `narrativa_gqm_pragmatic.md`: Narrativa explicativa del enfoque.
4. `mapeo_gqm_normativo.md`: Mapeo detallado de métricas a requisitos normativos.
5. `plantilla_excel_gqm_pragmatic.md`: Plantilla orientativa para implementación en Excel.
6. `plantilla_ppt_gqm_pragmatic.md`: Plantilla de reporte ejecutivo para PowerPoint.

## Cómo utilizarlo

- Utilice el marco para validar que cada métrica nacional responde a un objetivo claro y a una pregunta concreta.
- Aplique la matriz PRAGMATIC para evaluar periódicamente la calidad de cada indicador y priorizar mejoras.
- Apóyese en el mapeo normativo para alinear los indicadores con ENS, NIS2 y otros marcos aplicables.
- Implemente la plantilla de Excel para operacionalizar el cálculo de IGM sectoriales, scores PRAGMATIC y, si procede, indicadores de riesgo mitigado por euro invertido.
- Use la plantilla de PowerPoint para informar a decisores políticos, órganos de supervisión y otros actores relevantes sin abrumarles con tablas infinitas.

## Recomendaciones

- Revise anualmente el catálogo de métricas y su evaluación PRAGMATIC.
- Documente cambios en definiciones, fuentes de datos y métodos de cálculo.
- No acumule métricas por inercia: si una métrica no es predictiva, ni relevante, ni accionable, considere seriamente retirarla o rediseñarla.
