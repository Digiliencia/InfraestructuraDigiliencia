## III. DESIGN GUIDELINES

### Color Scheme (Professional)
```
Primary (Governance): Dark Blue #003D7A
Secondary (Risk): Red #D32F2F
Success (Progress): Green #388E3C
Neutral (Data): Gray #757575
Highlight (Urgent): Orange #F57C00

Usage:
├─ RED: Risks, gaps, things that need action
├─ GREEN: Achievements, positive progress, completed items
├─ BLUE: Governance, processes, policy
├─ ORANGE: Warnings, urgent items
└─ GRAY: Data, metrics, neutral information
```

### Typography
```
Title Slide: 60pt, bold, primary color
Section Headers: 44pt, bold, primary color
Content (body): 24-28pt, readable from 10+ feet away
Notes: 18pt, can be smaller (for speaker only)
Minimize text: Use visuals instead of bullet points where possible
```

### Layout
```
Standard 16:9 widescreen (1920×1080 minimum)
Margins: 1 inch all sides
White space: 30-40% of slide (not crowded)
One idea per slide (don't overload)
High contrast: Dark text on light background, or vice versa
```
