**Versión:** 1.0 | **Fecha:** Enero 2026

## 🎯 PROPÓSITO GENERAL DEL KIT

Proporcionar **framework integral, operacional y regulatorio** para evaluar madurez ciberseguridad de organizaciones españolas en 7 módulos (51 preguntas), alineado con:

- ✅ **Regulaciones:** NIS2 (Aug 2026), ISO 27001, GDPR, Plan Nacional
- ✅ **Metodología:** GQM (Goal-Question-Metric) + 5-level maturity
- ✅ **Aplicabilidad:** 7 módulos holísticos (gobernanza + técnico + compliance + cultura)
- ✅ **Implementación:** Roadmap 4 fases, presupuesto €2.1M, ROI 180%

### 1️⃣ DOCUMENTO 1: Modelo de Encuesta Integral

     - Madurez programa VM

     - Integración threat intelligence

     - MTTR vulnerabilidades críticas
     - Metodología testing
     - Retest post-remediation
     - % apps evaluadas
     - Scope testing elements

  3. **SIEM & Detección** (10 preguntas)
     - Existencia SIEM solution
     - Log sources integradas
     - Capacidad retención logs
     - Tipos alertas implementadas
     - Volumen eventos procesados
     - False positive rate
     - Integración SIEM-SOAR
     - Threat hunts ejecutadas
     - Frameworks cubiertos
     - Frecuencia reportería

  4. **Acceso e Identidad** (3 preguntas)
     - Sistema gestión identidades
     - MFA implementation
     - PAM program existence

  5. **Capacitación & Conciencia** (8 preguntas)
     - Tipos capacitación ofrecida
     - Frecuencia obligatoria
     - % completación
     - Medición resultados
     - Phishing simulation results
     - Comunicación a ejecutivos
     - Enfoques engagement
     - Tópicos cubiertos

  6. **Respuesta a Incidentes** (7 preguntas)
     - Plan IR formalizado
     - Playbooks documentadas
     - Frecuencia testing
     - Tiempo MTTR promedio
     - Integración con otros sistemas
     - Post-incident review
     - Threat intel integration

  7. **Estrategia Integrada** (9 preguntas)
     - Estrategia documentada
     - Alineación con negocio
     - Roadmap 3-year
     - Governance framework
     - Métricas clave definidas
     - Reporting cadence
     - Risk appetite statement
     - Resource allocation
     - Continuous improvement process

- **Formato:** Preguntas cerradas + abiertas (mix)
- **Escala:** Levels 0-5 (Inicial → Optimizado)

### 2️⃣ DOCUMENTO 2: Guía Metodológica Detallada
**Archivo:** `guia_metodologica_encuesta_es.md`  
**Tamaño:** ~25 páginas  
**Propósito:** **Instrucciones facilitador + scoring criteria**

#### Contenido:
- **Introducción GQM methodology:** 4-level hierarchy (Strategic → Tactical → Operational → Metrics)
- **Escala madurez 0-5:**
  - Nivel 0: Inexistente
  - Nivel 1: Inicial (ad-hoc)
  - Nivel 2: Repetible (process emerging)
  - Nivel 3: Definido (formal documented)
  - Nivel 4: Gestionado (measured & monitored)
  - Nivel 5: Optimizado (continuous improvement)

- **Por cada pregunta:**
  - Scoring guidance (0-5 rubric)
  - Examples de respuestas Level 1 vs Level 3 vs Level 5
  - Evidence requerida (data sources)
  - Red flags (common mistakes)
  - Probing questions (para esclarecer)

- **Scoring consolidation:** Cómo agregar respuestas (promedio vs weighted)
- **Validation techniques:** Cómo resolver divergencias >15pp
- **Quality assurance:** Checks post-assessment

#### Quién lo usa:
- **Facilitador:** Referencia durante interviews (scoring criteria)
- **Analyst:** Para validación/consolidation post-interviews
- **CISO:** Para entender escala interpretación

#### Cuándo se usa:
- **Pre-assessment:** Facilitador training (1-2 horas)
- **During:** Consulta scoring criteria
- **Post:** Validación scoring antes consolidación

   - Regulatory readiness heatmap

2. **Module Detail (7 hojas, 1 por módulo):**
   - Preguntas desglosadas
   - Individual scores
   - PRAGMATIC-style criteria (Predictive, Relevant, Actionable, etc.)
   - Fortalezas/gaps por módulo
   - Recomendaciones

3. **Costs & ROI:**
   - Presupuesto desglosado (€2.1M):
     - Phase 1: €0
     - Phase 2: €1.2M
     - Phase 3: €200K
     - Phase 4: €700K
   - ROI scenarios (Conservative/Realistic/Optimistic)
   - Payback period calculations
   - Sensitivity analysis

4. **Trajectory (18-36 meses):**
   - Quarter-by-quarter IMC projection
   - Module scores evolution
   - Phase milestones
   - Success criteria 2026-2027

5. **Raw Data Input:**
   - Field para entry datos brutos
   - Data sources specification
   - Monthly update sheet

#### Fórmulas Clave:
```excel
IMC = (Avg_Módulos × 0.50) + (Compliance% × 0.30) + (Implementation% × 0.20)
ROI% = (Benefits - €2.1M) / €2.1M × 100
Payback (meses) = 12 / (Annual_Benefit/12 / €2.1M)
```
