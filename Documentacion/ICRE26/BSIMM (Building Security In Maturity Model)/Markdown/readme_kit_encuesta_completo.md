## ¿QUÉ ES ESTE KIT?

Un **sistema integral y profesional** para evaluar, medir y mejorar la madurez de ciberseguridad de tu organización en España 2026, utilizando:

- 🎯 **68 Preguntas Estructuradas**: Cobertura completa de 8 módulos
- ✅ **Mapeo Normativo Completo**: NIS2, CRA, GDPR, ISO 27001, ENS
- 📊 **Cálculo Automático IGM**: Índice Gestión Madurez 0-100
- 💰 **Análisis Financiero**: ROI, presupuesto, beneficios
- 📋 **Templates Operativos**: Excel, PowerPoint, Checklists

**Objetivo Final**: Alcanzar IGM 70 (Operativo/L3) cumplimiento NIS2 en 12 meses

## CÓMO USAR ESTE KIT

### Escenario 1: "Soy CISO, necesito ejecutar encuesta YA"

**Paso 1: Lectura Rápida (1 hora)**
```
1. Lee este README (15 min)
   → Entiende estructura kit

2. Escanea NARRATIVA EXPLICATIVA (15 min)
   → Aprecia estilo y propósito

3. Revisa ENCUESTA INTEGRAL (30 min)
   → Familiarízate con 68 preguntas

4. Decide: ¿Proceder sí/no?
```

**Paso 2: Preparación (2-3 horas)**
```
1. Imprime ENCUESTA INTEGRAL (67 páginas A4)
   → O distribuye versión digital

2. Copia PLANTILLA EXCEL (setup 6 hojas)
   → Prepara para respuestas

3. Reúnete con CISO + Steering Committee (1 h)
   → Aprueba timeline, responsables

4. Resultado: Kit listo para ejecutar
```

**Paso 3: Ejecución (2-3 semanas)**
```
1. Semana 1: Kick-off + Distribución
   - Presenta propósito encuesta
   - Distribuye a owners (68 preguntas)
   - Recopila respuestas

2. Semana 2: Consolidación
   - Ingresa respuestas en EXCEL
   - Calcula IGM automático
   - Identifica brechas por módulo

3. Semana 3: Análisis + Reporte
   - Crea PPT usando PLANTILLA
   - Presenta a Dirección (20 min)
   - Aprueba roadmap Q2
```

**Paso 4: Seguimiento (Permanente)**
```
1. Mapea acciones a MAPEO NORMATIVO
   → Cada brecha conecta a requisito NIS2/GDPR

2. Actualiza EXCEL trimestralmente
   → Mide progreso vs baseline

3. Presenta a Board cada trimestre
   → Usa PLANTILLA POWERPOINT

4. Objetivo: 45.2 → 70 en 12 meses
```

```

   - SIEM → SOC Manager
   - Respuesta Incidentes → IR Lead
```

```
   - IGM Score (Hoja 2)
   - ROI por iniciativa (Hoja 3)
   - KPIs (Hoja 4)
   - Budget vs actual (Hoja 5)

3. Genera gráficos automáticos
   - Trending, Speedometer, Budget, Progress, Radar
```

**Paso 4: Reporte (Semana 3)**
```
1. Copia datos Excel → PLANTILLA POWERPOINT
   - Slides 4-12 auto-actualizan

2. Personaliza narrativa (2-3 h)
   - SLIDE 2: Executive Summary
   - SLIDE 3: Goals
   - SLIDE 9: Q2 Roadmap

3. Presenta a Board (20 min)
```

## FAQ (PREGUNTAS FRECUENTES)

### P1: ¿Debo responder todas 68 preguntas?

**R**: Sí, pero pueden ser respondidas por diferentes personas:
- Gobernanza (5Q) → CISO
- Activos (5Q) → CTO
- Vulnerabilidades (8Q) → Security Team
- Testing (5Q) → QA Team
- SIEM (5Q) → SOC
- Respuesta Incidentes (7Q) → IR Team
- Capacitación (6Q) → HR/Training
- Supply Chain (5Q) → Procurement

Tiempo total: ~2-3 horas distribuidas

---

### P2: ¿Qué escala de respuesta uso?

**R**: 
- SÍ = Implementado operativamente
- PARCIAL = Parcialmente implementado
- NO = No implementado

En EXCEL: SÍ=1, PARCIAL=0.5, NO=0

---

### P3: ¿Qué IGM Score significa?

**R**:
- 0-20: L0 (Inexistente) ❌
- 21-40: L1 (Ad-hoc) ❌
- 41-60: L2 (Documentado) ⚠️
- **61-80: L3 (Operativo)** ✅ **← NIS2 Target**
- 81-95: L4 (Gestionado) ✅
- 96-100: L5 (Optimizado) ✅

---

### P4: ¿Cuánto tiempo toma la encuesta?

**R**:
- Preparación: 2-3 horas
- Ejecución: 2-3 horas distribuidas
- Análisis: 2-3 horas
- Reporte: 2-3 horas
- **Total: 8-12 horas de trabajo**

Puede comprimirse a 1 semana si equipos trabajan en paralelo

---

### P5: ¿Puedo hacer esto sin herramientas especiales?

**R**: Sí. Mínimo necesario:
- ✅ Excel o Google Sheets (para PLANTILLA EXCEL)
- ✅ PowerPoint o Google Slides (para PLANTILLA PPT)
- ✅ Google Forms o Survey Monkey (para ENCUESTA opcional)

No requiere software costoso

---

### P6: ¿Cada cuánto se repite?

**R**:
- **Encuesta Completa**: Anuales (Enero)
- **Encuesta Lite** (solo brechas): Trimestrales
- **Dashboard IGM**: Mensual actualización
- **Reporte Board**: Trimestral

---

### P7: ¿Cómo valido contra regulatorio?

**R**:
1. Consulta MAPEO ENCUESTA NORMATIVO
2. Cada SÍ debe tener evidencia:
   - Política formal (documento)
   - Proceso (procedimiento)
   - Sistema (herramienta)
   - Log (auditoría)

---

### P8: ¿Aplica para PYME?

**R**: Sí, pero simplificado:
- PYME: 30 preguntas prioritarias (50% encuesta)
- Sectoriales: +10 preguntas según industria
- Timeline: 6 meses vs 12 meses enterprise
