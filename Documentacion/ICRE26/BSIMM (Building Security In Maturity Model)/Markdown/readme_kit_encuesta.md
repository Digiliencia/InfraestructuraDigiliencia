## ¿QUÉ ES ESTE KIT?

Un **sistema integral y profesional** para evaluar, medir y mejorar la madurez de seguridad de software en tu organización, usando:

- 🎯 **GQM** (Goal-Question-Metric): Metodología de trazabilidad
- ✅ **PRAGMATIC**: 9 criterios para validar utilidad de métricas
- 📊 **BSIMM15**: Base de 121 organizaciones, 128 actividades
- 📋 **Templates**: Excel, PowerPoint, Markdown, Checklist

**Objetivo Final**: Alcanzar 70 puntos NIS2 en 12 meses (Q1-Q4 2026)

### Escenario 1: "Soy CISO, necesito implementar BSIMM en Q1"

**Paso 1: Lectura Rápida (1 hora)**
```
1. Lee RESUMEN EJECUTIVO (15 min)

```

**Paso 4: Ejecución (90 días)**

### Escenario 3: "Soy Auditor, necesito validar NIS2"

**Paso 1: Mapeo Regulatorio (1 hora)**
```
1. Lee MAPEO GQM NORMATIVO (30 min)
   - Cómo cada Pregunta GQM conecta a NIS2 Medidas
   - Cobertura: 8/10 medidas (80%)

2. Obtén lista de PREGUNTAS GQM implementadas (30 min)
   - Ask CISO: ¿Cuáles de las 30 preguntas están respondidas?
   - Mapea a tu checklist de compliance
```

**Paso 2: Validación de Implementación**
```
Para cada Pregunta GQM implementada:
☐ ¿Existe política/procedimiento formal documentado?
☐ ¿Se ejecuta operativamente (más de 1 instancia)?
☐ ¿Se ha auditado internamente el último año?
☐ ¿Se han cerrado hallazgos previos?
☐ ¿Alinea a requisito NIS2/ENS/GDPR específico?

Si 5/5: COMPLIANT ✅
Si 3-4/5: PARCIAL ⚠️
Si <3/5: NO COMPLIANT ❌
```

**Paso 3: Opinión de Auditoría**
```
Reporte:
"Evaluación BSIMM15 + GQM + PRAGMATIC muestra:

✅ Implementación de X/30 preguntas GQM
✅ Cobertura NIS2: 8/10 medidas (80%)
✅ Status: Camino correcto a compliance 2026

⚠️ Brechas pendientes:
   - Medida 5 (Criptografía): Parcial
   - Medida 6 (Backup): No evaluado
   
RECOMENDACIÓN: Plan de remediación en Q2-Q3"
```

### Escenario 4: "Soy Implementador, necesito pasos concretos"

**Paso 1: Preparación (Semana 1)**
```
1. Descarga CUADERNO DE TRABAJO
2. Lee Parte I (GQM): 30 min
3. Lee Parte II (PRAGMATIC): 30 min
4. Reúnete con CISO: Clarifica GOAL local
```

**Paso 2: Evaluación (Semana 2-3)**
```
1. Completa Plantilla GQM (Parte I Cuaderno)
   - Llena GOAL, QUESTIONS, METRICS
   - Resultado: ~10 métricas clave

2. Completa Plantilla PRAGMATIC (Parte II Cuaderno)
   - Evalúa cada métrica vs 9 criterios
   - Califica 0-100 cada criterio
   - Suma: Score PRAGMATIC por métrica

3. Resultado: Matriz GQM + PRAGMATIC para tu org
```

**Paso 3: Priorización (Semana 4)**
```
1. Selecciona PRAGMATIC ≥85 (Quick Wins Q1)
   - Implementar en 90 días
   - Presupuesto ~€25k

2. Aparta PRAGMATIC 70-84 (Baseline Q2-Q3)
   - Implementar con CISO validation
   - Presupuesto ~€70k

3. Descarta PRAGMATIC <70 (o aplaza Q4+)
```

**Paso 4: Implementación (Semana 5+)**
```
Para cada actividad TOP 10:
1. Designa propietario (owner)
2. Estima duración (típicamente 30-45 días)
3. Asigna presupuesto
4. Crea tickets en Jira/Azure DevOps
5. Sigue CUADERNO Parte IV (Roadmap)

Medir cada 2 semanas:
- % completación
- Presupuesto gastado
- Riesgos emergentes
```

**Paso 5: Tracking (Permanente)**
```
1. Usa PLANTILLA EXCEL para tracking:
   - Hoja 1: Datos entrada (actividades + status)
   - Hoja 2: IGM Calculation (score automático)
   - Hoja 3: ROI (beneficios)
   - Hoja 4: Dashboard (KPIs)

2. Reporta mensualmente a CISO:
   - % avance vs plan
   - Presupuesto vs actual
   - Riesgos y mitigaciones

3. Reporte trimestral a Dirección:
   - Usa PLANTILLA POWERPOINT (12 slides)
   - 20 minutos presentación
   - Incluye recomendaciones next quarter
```

### P2: ¿Cuánto tiempo toma implementar TODO?

**R**:

- **Implementación Q1**: 90 días
- **Reporting**: 4-6 horas por trimestre

**Total Año 1**: ~40-50 horas gestión + 90 días ejecución

### P3: ¿Qué pasa si mi presupuesto es menor a €258k?

**R**: 
1. Implementa SOLO TOP 4 actividades Q1 (€26k)
   - Resultado: Score 56 → 71 (+15 puntos)
   - ROI: 1,515%

2. Aplaza Q2-Q4 para años siguientes

3. Expected end-2026: Score 71 (vs. target 70) ✅

---

### P4: ¿Cómo valido que las métricas son correctas?

**R**: 
1. Cada métrica tiene PRAGMATIC score (0-100)
2. Si <70: Rechaza o reemplaza
3. Si ≥85: Adopta inmediatamente
4. Si 70-84: Evalúa ROI antes de adoptar

Ver MATRIZ PRAGMATIC COMPLETA para evaluación de cada actividad.

---

### P5: ¿Qué pasa después de Q1?

**R**:
1. Reporte a Dirección (PLANTILLA POWERPOINT)
2. Lecciones aprendidas (CUADERNO Parte VIII)
3. Rebaselining (PLANTILLA EXCEL)
4. Plan Q2 (del RESUMEN EJECUTIVO)
5. Repetir ciclo

---

### P6: ¿Aplica este kit a otras organizaciones?

**R**: SÍ.
- GQM es agnóstico (usa PRAGMATIC para evaluar cualquier métrica)
- BSIMM15 es estándar global (121 orgs, 15 años datos)
- NIS2/ENS/GDPR es regulatorio España (adaptable otras regiones)

**Personalización**: Usa CUADERNO TRABAJO Parte I para customizar GOAL local.

### P7: ¿Quién debe estar en steering committee?

**R**: 
- ✅ CISO (propietario programa)
- ✅ CTO/VP Engineering (implementación técnica)
- ✅ CFO/Analyst (financiero)
- ✅ Compliance/Risk officer
- ✅ Principales application owners

**Reuniones**: Bi-semanal (30 min status, 30 min riesgos)

### P8: ¿Cómo reporto progreso a auditoría?

**R**:
1. Usa MAPEO NORMATIVO para trazabilidad → NIS2/GDPR
2. Referencia PREGUNTA GQM específica implementada
3. Proporciona evidencia (documento, métrica, logs)
4. Valida que PRAGMATIC ≥70 para cada métrica

Ver MAPEO NORMATIVO sección "Auditoría: Validación de Cumplimiento"
