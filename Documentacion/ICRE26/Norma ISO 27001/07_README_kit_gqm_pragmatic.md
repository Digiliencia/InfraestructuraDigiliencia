# README – Kit de Encuesta e Indicadores ISO 27001 con GQM + PRAGMATIC

## 1. Qué es este kit

Este kit amplía el “Kit de Encuesta ISO 27001” incorporando:

- Un **marco GQM** (Goal‑Question‑Metric) para garantizar trazabilidad entre objetivos nacionales, objetivos del SGSI y métricas técnicas. [web:40][web:43][web:51]
- Una **evaluación PRAGMATIC** de las métricas (Predictive, Relevant, Actionable, Genuine, Meaningful, Accurate, Timely, Independent, Cheap) para priorizar indicadores útiles y sostenibles. [web:44][web:49][web:52]

## 3. Cómo usarlo en tres pasos

1. **Diseñar/ajustar la encuesta y el catálogo de métricas**  
   - Partir del cuestionario ISO 27001 ya definido.  
   - Seleccionar un subconjunto de métricas clave (M‑*) alineadas con objetivos GQM.  
   - Revisar su puntuación PRAGMATIC y ajustar o descartar las que suspendan claramente.

2. **Recoger datos y calcular subíndices e IGM**  
   - Volcar respuestas y datos técnicos en la hoja de cálculo.  
   - Calcular subíndices por objetivo GQM y el IGM global.  
   - Revisar la contribución de cada métrica basada en su peso PRAGMATIC.

3. **Comunicar y decidir**  
   - Preparar el reporte ejecutivo.  
   - Presentar resultados y, sobre todo, las prioridades de mejora.  
   - Definir un plan de acción y revisarlo periódicamente.

## 4. Buenas prácticas sugeridas

- **Mantener un catálogo compacto de métricas**: mejor 20 buenas que 80 olvidadas.  
- **Revisar periódicamente las puntuaciones PRAGMATIC**: las métricas envejecen, el contexto cambia.  
- **Documentar la cadena GQM**: para cada métrica clave, dejar claro a qué objetivo sirve y qué pregunta responde.  
- **Usar los resultados para quitar métricas inútiles**, no solo para añadir más.

## 5. Evolución del kit

Este kit está pensado para evolucionar con:

- Nuevas versiones de estándares (ISO, NIS2, etc.).  
- Cambios en las prioridades nacionales y sectoriales (por ejemplo, foco en OT, resiliencia de cadenas de suministro).  
- Lecciones aprendidas de su aplicación en organizaciones reales.

Si con el tiempo descubres que algunos indicadores apenas miran la luz y otros se convierten en referencias habituales en comités de dirección, ya tienes una pista clara de qué conservar y qué jubilar con honores.
