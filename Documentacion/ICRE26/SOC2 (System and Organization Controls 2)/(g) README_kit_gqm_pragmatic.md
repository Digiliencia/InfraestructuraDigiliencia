# (g) README – Kit GQM + PRAGMATIC para indicadores SOC 2 – ENS – NIS2

Este kit complementa el cuestionario de madurez SOC 2 – ENS – NIS2 con un marco de métricas trazables (GQM) y de alta calidad (PRAGMATIC).

## Archivos incluidos

- **(a) marco_integrativo_gqm_pragmatic.md**: Explica el marco conceptual, con ejemplos por bloques SOC 2.
- **(b) matriz_evaluacion_pragmatic.md**: Resumen tabular de la evaluación PRAGMATIC para métricas clave.
- **(c) narrativa_gqm_pragmatic.md**: Texto explicativo en tono divulgativo para compartir con dirección y equipos.
- **(d) mapeo_gqm_normativa.md**: Tabla que vincula objetivos, preguntas y métricas con SOC 2, ENS, NIS2 y RGPD.
- **(e) plantilla_excel_gqm_pragmatic_igm_roi.md**: Estructura sugerida de hoja de cálculo para implementar el modelo.
- **(f) plantilla_reporte_ejecutivo_gqm_pragmatic.md**: Guion de presentación para exponer resultados y decisiones.

## Uso sugerido

1. Seleccionar los objetivos prioritarios (G) para la organización.
2. Validar las preguntas (Q) que realmente interesan a la dirección y a los reguladores.
3. Elegir métricas (M) que superen un umbral mínimo PRAGMATIC (por ejemplo, ≥ 18/27).
4. Implementar la captura de datos en la plantilla de Excel.
5. Integrar los resultados en el cálculo del IGM y en la narrativa de ROI.
6. Comunicar hallazgos usando la plantilla de reporte ejecutivo.

Si una métrica no sabe explicar a qué objetivo sirve ni aprueba el examen PRAGMATIC, quizá te está midiendo más a ti que a tu ciberseguridad.
