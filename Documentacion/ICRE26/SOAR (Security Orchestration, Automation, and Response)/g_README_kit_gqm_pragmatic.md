# README – Kit GQM + PRAGMATIC para indicadores SOAR

Este kit complementa el Kit de Encuesta SOAR incorporando un marco explícito de trazabilidad (GQM) y calidad de métricas (PRAGMATIC) para el diseño de tableros nacionales y corporativos.[web:22][web:24]

## Archivos incluidos

- **a_marco_integrativo_gqm_pragmatic.md** – Descripción del marco GQM + PRAGMATIC aplicado a indicadores SOAR nacionales.
- **b_matriz_evaluacion_pragmatic_completa.md** – Tabla de evaluación PRAGMATIC para las principales métricas propuestas.
- **c_narrativa_explicativa_gqm_pragmatic.md** – Texto explicativo para acompañar la adopción del marco en foros directivos y técnicos.
- **d_mapeo_preguntas_normativa_gqm_pragmatic.md** – Relación entre preguntas de encuesta, métricas GQM y marcos normativos.
- **e_plantilla_excel_igm_roi_gqm_pragmatic.md** – Guía para implementar en Excel el cálculo de IGM y ROI ajustados por GQM + PRAGMATIC.
- **f_plantilla_reporte_ejecutivo_gqm_pragmatic.md** – Estructura de presentación ejecutiva centrada en objetivos, métricas y calidad de indicadores.

## Cómo usar este kit

1. **Diseñar o revisar el cuestionario SOAR**
   - Asegúrese de que cada bloque de preguntas se vincula claramente a un objetivo G1–G4 y a una o más métricas M1.x–M4.x.

2. **Seleccionar el subconjunto de métricas prioritarias**
   - Utilice la matriz PRAGMATIC para elegir las métricas que ocuparán el “primer nivel” del tablero de mando.

3. **Construir la plantilla de análisis**
   - Implemente la estructura de hojas de Excel propuesta, alineando los cálculos con GQM y aplicando pesos PRAGMATIC cuando proceda.

4. **Comunicar resultados con contexto**
   - Use la plantilla de reporte ejecutivo para presentar los resultados, enfatizando la trazabilidad (objetivos → métricas) y la calidad de los indicadores, no solo sus valores.

## Personalización

El marco y las plantillas se pueden adaptar para:

- Sectores específicos (sanidad, energía, finanzas, OT). 
- Ámbitos territoriales (comunidades autónomas, ciudades). 
- Tipos de organizaciones (pymes, grandes grupos, administraciones públicas).

La recomendación es mantener la lógica GQM + PRAGMATIC intacta y personalizar principalmente el contenido de objetivos, preguntas y métricas concretas.
