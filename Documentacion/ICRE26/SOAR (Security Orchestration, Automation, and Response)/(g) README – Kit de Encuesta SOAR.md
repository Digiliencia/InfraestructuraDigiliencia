# README – Kit de Encuesta SOAR

Bienvenido/a al kit de materiales para la Encuesta Nacional sobre capacidades SOAR (Security Orchestration, Automation and Response).  
Este repositorio reúne todo lo necesario para desplegar, analizar y comunicar los resultados de la encuesta en organizaciones españolas.

## Contenidos

- **a_modelo_encuesta.md** – Cuestionario completo en formato Markdown, listo para implementarse en una herramienta de encuestas online o adaptarse a otros formatos.  
- **b_guia_metodologica.md** – Descripción detallada de objetivos, población objetivo, diseño del cuestionario y enfoque analítico, incluyendo el Indicador Global de Madurez (IGM).  
- **c_narrativa_explicativa.md** – Texto explicativo para acompañar la encuesta y contextualizarla ante los participantes, con un tono divulgativo y ligeramente irónico.  
- **d_mapeo_normativo.md** – Tabla que relaciona cada bloque de preguntas con marcos normativos y de buenas prácticas (NIS2, ENS, NIST, ENISA).  
- **e_plantilla_excel_igm_roi.md** – Instrucciones para construir una plantilla de Excel que calcule IGM y estime el ROI de SOAR a partir de las respuestas.  
- **f_plantilla_reporte_ejecutivo.md** – Guion para una presentación ejecutiva en PowerPoint con los resultados agregados.  

## Uso recomendado

1. **Adaptar el cuestionario**  
   Revise `a_modelo_encuesta.md` y ajuste la terminología a su contexto sectorial o institucional (por ejemplo, referencias a CSIRT sectoriales propios).

2. **Configurar la herramienta de encuesta**  
   Implemente las preguntas en la plataforma elegida, respetando en lo posible las escalas propuestas para facilitar el cálculo posterior del IGM y el ROI.

3. **Informar a los participantes**  
   Utilice la `c_narrativa_explicativa.md` (o una versión adaptada) como acompañamiento del correo de invitación, para que el propósito y el tono de la encuesta queden claros.

4. **Procesar las respuestas**  
   Exporte los resultados a un formato compatible con Excel o herramienta de análisis equivalente.  
   Siga las indicaciones de `e_plantilla_excel_igm_roi.md` para estructurar las hojas y fórmulas de cálculo.

5. **Elaborar informes y presentaciones**  
   Aplique la estructura propuesta en `f_plantilla_reporte_ejecutivo.md` para preparar presentaciones a equipos directivos, reguladores o foros sectoriales.

## Consideraciones de privacidad y confidencialidad

- Asegúrese de que la encuesta y el tratamiento de datos cumplen con las obligaciones de confidencialidad y protección de datos aplicables (RGPD, normativa interna).  
- Se recomienda trabajar con datos agregados y anonimizados para la difusión de resultados externos.

## Personalización y extensiones

El kit está diseñado para servir como plantilla base.  
Es perfectamente legítimo (y recomendable) que:

- Añada módulos específicos para sectores concretos (sanidad, energía, banca).  
- Incorpore preguntas sobre integración con marcos internos de continuidad de negocio y gestión de crisis.  
- Conecte los resultados con programas de formación o iniciativas de mejora continua.

Si el uso de este kit le inspira una nueva hornada de dudas, propuestas o epifanías sobre SOAR, considérelo una funcionalidad de serie y no un bug.
