# MATRIZ DE EVALUACIÓN PRAGMATIC COMPLETA

**Evaluación de 128 Actividades BSIMM15 contra 9 Criterios PRAGMATIC**

**Versión**: 1.0  
**Fecha**: Enero 21, 2026  
**Cobertura**: 121 organizaciones globales  
**Escala**: 0-100 por criterio (Promedio = Score PRAGMATIC)

---

## LEYENDA

| Símbolo | Significado |
|---------|-----------|
| **P** | Predictive: ¿Predice riesgo futuro? |
| **R** | Relevant: ¿Es relevante a objetivos negocio? |
| **A** | Actionable: ¿Lleva a acción específica? |
| **G** | Genuine: ¿Basada en datos reales (no teóricos)? |
| **M** | Meaningful: ¿Tiene significado operacional? |
| **A** | Accurate: ¿Es medible y precisa? |
| **T** | Timely: ¿Se obtiene en tiempo útil? |
| **I** | Independent: ¿No depende de otras métricas? |
| **C** | Cost-effective: ¿Económicamente viable? |

**Colores**:
- 🟢 Green (85-100): Adoptar inmediatamente
- 🟡 Yellow (70-84): Evaluar y adoptar
- 🔴 Red (<70): Bajo pragmatismo, considerar alternativas
