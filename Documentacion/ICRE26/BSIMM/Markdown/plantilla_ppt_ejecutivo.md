# PLANTILLA REPORTE EJECUTIVO: PRESENTACIÓN POWERPOINT

**Guía para Crear Presentación Ejecutiva en PowerPoint**

**Versión**: 1.0  
**Fecha**: Enero 21, 2026  
**Formato**: Especificación Markdown para PowerPoint/Google Slides  
**Objetivo**: Presentación trimestral a Dirección (20 min)

---

## INTRODUCCIÓN

Este documento describe la estructura, contenido y diseño de una presentación PowerPoint para reportar progreso GQM+PRAGMATIC a dirección ejecutiva.

**Duración**: 20 minutos  
**Audiencia**: CEO, CFO, Board Members  
**Secciones**: 12 slides + Apéndice  
**Frecuencia**: Trimestral (Q1, Q2, Q3, Q4)

---

## INSTRUCCIONES DE DISEÑO POWERPOINT

### Paleta de Colores

```
Primarios:
- Azul corporativo: #003366 (headers, accent)
- Blanco: #FFFFFF (fondo principal)
- Gris oscuro: #333333 (texto)

Secundarios:
- Verde éxito: #00AA44 (completado, on-track)
- Amarillo advertencia: #FFAA00 (en progreso, warning)
- Rojo crítico: #DD2222 (problema, off-track)

Acentos:
- Azul claro: #0099CC (datos, gráficos)
- Gris claro: #EEEEEE (background subtítulos)
```

### Tipografía

```
Títulos: Calibri Bold, 44 pt, Color #003366
Subtítulos: Calibri Regular, 28 pt, Color #333333
Body: Calibri Regular, 18 pt, Color #333333
Tablas: Calibri Regular, 14 pt, Color #333333

Line spacing: 1.5
Margin: 1 pulgada todos lados
```

### Elementos Gráficos

```
Iconos: Material Design (check, arrow, warning, etc.)
Gráficos: Office charts con colores corporativos
Imágenes: Compression 70%, Border 1px gris
Logos: Máximo 2" de ancho, opacity 40% si background

Transiciones: Fade (0.5 seg)
Animaciones: Entrance (Appear) para bullet points
```

## INSTRUCCIONES DE PRESENTACIÓN

### Timing

```
Total: 20 minutos
- Slides 1-3: 3 min (contexto)
- Slides 4-7: 8 min (progreso, finanzas)
- Slides 8-10: 5 min (riesgos, logros)
- Slides 11-12: 4 min (recomendaciones, cierre)
- Q&A: 5-10 min reserve
```

### Técnica de Presentación

```
1. Comienza con Slide 2 (Executive Summary) - catcha atención
2. Usa apuntador para señalar gráficos específicos
3. Lee números directos de tablas (credibilidad)
4. Permita interrupciones en secciones Financiero y Riesgos
5. Cierra con llamado a acción (decisiones ejecutivas)
```

### Manejo de Preguntas

```
"How can we ensure 70 point target?"
→ Refiere a Slide 5 (timeline) + Slide 7 (ROI justifica inversión)

"What if adoption rate drops?"
→ Refiere a Slide 8 (riesgos ya identificados + mitigación)

"Why this approach vs. alternatives?"
→ Refiere a Slide 4 (TOP 10 PRAGMATIC - evidence-based)

"Timeline to compliance?"
→ Refiere a Slide 3 (NIS2 80% cubierto ya, 100% Q2-Q3)
```

## EXPORTACIÓN Y DISTRIBUCIÓN

### Formatos

- **PowerPoint**: PPTX (editable)
- **PDF**: Para archivado y distribución segura
- **Google Slides**: Para collaboration en tiempo real
- **Video**: Grabar presentación + salvar como MP4

### Distribución

1. Enviar PPTX a Dirección 1 día antes de meeting
2. Presentar en vivo con proyector/TV
3. Distribuir PDF después de presentación
4. Guardar versión final en SharePoint (control de versiones)
