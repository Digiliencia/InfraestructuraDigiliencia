# (g) README – Kit GQM + PRAGMATIC CTEM

Este README describe el contenido y uso del kit de documentos para aplicar el marco GQM + PRAGMATIC a indicadores CTEM.

## 1. Contenido del kit

- (a) **Marco integrativo GQM + PRAGMATIC**: define objetivos, preguntas y métricas CTEM a nivel nacional, y cómo aplicar PRAGMATIC.
- (b) **Matriz de Evaluación PRAGMATIC Completa**: tabla con métricas M1.x–M6.x y puntuaciones orientativas.
- (c) **Narrativa explicativa**: contexto y justificación del enfoque.
- (d) **Mapeo detallado de métricas a requisitos normativos**: tabla enlazando cada métrica con NIS2, DORA, ISO, NIST.
- (e) **Plantilla de Excel para IGM y ROI**: guía para implementar los cálculos en hoja de cálculo.
- (f) **Plantilla de Reporte Ejecutivo en PowerPoint**: estructura sugerida de presentación de resultados.

## 2. Cómo usar el kit

1. Adapte el marco GQM a sus objetivos específicos (nacionales, sectoriales u organizativos).
2. Revise las métricas propuestas y su evaluación PRAGMATIC; elimine o ajuste las que no encajen.
3. Implante la recogida de datos necesaria (encuestas, integraciones CTEM, repositorios nacionales).
4. Construya la plantilla de Excel siguiendo el documento (e) y calcule IGM/ROI.
5. Presente y discuta resultados con la plantilla de reporte ejecutivo.

## 3. Principios de diseño

- Pocas métricas bien elegidas y evaluadas superan a catálogos interminables.
- La trazabilidad de GQM permite explicar claramente por qué se mide algo.
- PRAGMATIC asegura que lo medido merezca la pena.

## 4. Mantenimiento y evolución

- Revise objetivos y métricas al menos cada 2 años.
- Actualice el mapeo normativo cuando cambien NIS2, DORA o sus desarrollos nacionales.
- Documente cambios significativos en una sección de control de versiones (no incluida aquí por brevedad).

Este kit está pensado para servir tanto a equipos nacionales/sectoriales como a organizaciones individuales que quieran alinear sus métricas CTEM con objetivos de alto nivel sin perder el contacto con la realidad técnica.
