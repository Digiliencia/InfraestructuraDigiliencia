## ¿QUÉ ES CRA-CYBER?

CRA-Cyber es un **kit integral de evaluación de madurez de ciberseguridad** diseñado para organizaciones españolas que necesitan:

✓ **Cuantificar costes reales** de ciberseguridad (metodología CAsPeA)  
✓ **Evaluar madurez** comparada con pares sectoriales (benchmarking)  
✓ **Demostrar ROI** de inversiones en seguridad  
✓ **Cumplir regulación** (NIS2, GDPR, ENS, DORA, ISO 27001)  
✓ **Justificar presupuestos** ante CFO, Junta Directiva, Auditores

## INSTRUCCIONES DE IMPLEMENTACIÓN

### FASE 1: PREPARACIÓN (Semanas 1-2)

**Paso 1.1: Descargar kit completo**
```
Descargar todos 7 documentos .md desde repositorio
Guardar en carpeta: /CRA-Cyber_v2.1/
```

**Paso 1.2: Configurar encuesta online**
```
OPCIÓN A — Google Forms (Gratuito):
  1. Crear nuevo formulario en Google Forms
  2. Copiar preguntas de Documento 1
  3. Configurar opciones (validación, requeridas)
  4. Generar enlace público

OPCIÓN B — Qualtrics (Profesional):
  1. Importar archivo XLS generado desde Documento 1
  2. Configurar branding y seguridad (HTTPS, SSO)
  3. Establecer permisos de respondientes

OPCIÓN C — PDF Impreso:
  1. Exportar Documento 1 a PDF
  2. Distribuir físicamente
  3. Digitalizar respuestas en Excel para análisis
```

**Paso 1.3: Preparar comunicación**
```
Email de invitación (ver Documento 2, Fase 1)
FAQ preelaboradas
Contacto de soporte técnico
```

### FASE 2: LANZAMIENTO (Semana 3)

**Paso 2.1: Invitar participantes**
```
Muestra recomendada: 150-250 respondientes
Distribución: 7-8 sectores, 20-30 por sector
Plazo: 3-4 semanas

Canales de difusión:
- INCIBE (directos contactos miembros)
- Cámaras de Comercio españolas
- Asociaciones sectoriales
- LinkedIn targeting
```

**Paso 2.2: Recordatorios**
```
Día 7: Recordatorio automatizado
Día 14: Recordatorio personal a respondientes clave
Día 21: Invitación final antes de cierre
```

### FASE 3: ANÁLISIS (Semanas 6-8)

**Paso 3.1: Descargar datos**
```
Exportar respuestas de Google Forms/Qualtrics a Excel
Realizar limpieza inicial (búsqueda de blancos, outliers)
```

**Paso 3.2: Cargar en plantilla Excel**
```
1. Abrir Documento 5 (Plantilla Excel)
2. Copiar respuestas a Hoja 2 (Datos brutos)
3. Ejecutar validaciones automáticas
4. Revisar Dashboard (Hoja 8)
```

**Paso 3.3: Análisis estadístico**
```
Usar Documento 2 (Guía Metodológica) como referencia
Calcular:
- Estadística descriptiva por pregunta
- IMG (Índice Madurez General)
- Comparativa sectorial (benchmarking)
- Análisis de brechas
- ROI cuantificado
```

**Paso 3.4: Generar reportes**
```
Usar Excel para exportar:
- Resumen ejecutivo (1 página)
- Informe detallado (10-15 páginas)
- Datos para auditoría (Excel detallado)
```

### FASE 4: PRESENTACIÓN (Semanas 9-10)

**Paso 4.1: Preparar presentación**
```
Usar Documento 6 (Plantilla PowerPoint)
Adaptar números a hallazgos reales
Ensayar timing (15-20 minutos)
```

**Paso 4.2: Presentar a stakeholders**
```
Audiencia: C-Suite, Junta Directiva, Auditores, Reguladores
Foco: IMG, ROI, recomendaciones prioritarias
Q&A preparada (ver Documento 6)
```

## FAQ — PREGUNTAS FRECUENTES

### P: ¿Cuánto tiempo toma completar la encuesta?
R: 45-60 minutos para respondiente autoadministrado. Si es entrevista guiada, 90 minutos.

### P: ¿Puedo hacer la encuesta en inglés?
R: Actualmente solo disponible en español. Traducción al inglés planeada para Q2 2026.

### P: ¿Los datos son confidenciales?
R: Sí. Datos completamente anonimizados. Solo análisis agregado por sector se reporta públicamente.

### P: ¿Cómo debo interpretar el IMG (Índice Madurez General)?
R: Ver Documento 2 (Guía Metodológica), sección "Métricas de Madurez". Rango 0-100, donde 60+ = Tier 3 (Defined), objetivo recomendado.

### P: ¿Qué pasa si mi organización tiene IMG muy bajo (<20)?
R: Es crítico. Recommends contratación de consultor externo para evaluación profunda. Riesgo regulatorio y operacional alto.

### P: ¿Puedo usar solo partes del kit?
R: Sí, pero se recomienda completitud. Mínimo recomendado: Documentos 1, 5, 6 (Encuesta + Cálculos + Presentación).

### P: ¿Cada cuánto debo repetir la encuesta?
R: Anualmente mínimo. Recomendado: Semestral para seguimiento de progreso post-implementación de mejoras.
